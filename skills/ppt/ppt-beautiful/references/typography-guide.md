# 字体选择与排版指南

> 来源：《PPT，要你好看》 - 邵云蛟
> 核心理念：字体是PPT的「声音」，选对字体等于选对了表达方式

---

## 一、字体选择原则

### 核心规则
1. **全篇不超过3种字体**（标题+正文+数据/强调）
2. **同一类信息用同一字体**
3. **中英文分别选择**
4. **优先使用系统自带字体**（避免兼容性问题）

### 字体性格对照表

| 字体类别 | 性格特征 | 适用场景 | 代表字体 |
|----------|----------|----------|----------|
| 黑体系 | 现代、专业、稳重 | 商务汇报、科技 | 思源黑体、微软雅黑 |
| 宋体系 | 传统、正式、文化 | 政务、文化、学术 | 思源宋体、方正小标宋 |
| 圆体系 | 亲和、活泼、轻松 | 教育、儿童、休闲 | 思源圆体、苹方 |
| 手写体 | 个性、创意、温馨 | 创意展示、个人品牌 | 仅用于装饰，少量使用 |

## 二、推荐字体组合

### 组合1：经典商务

```yaml
中文标题: "思源黑体 Bold"       # 或 "阿里巴巴普惠体 Bold"
中文正文: "思源黑体 Regular"    # 或 "微软雅黑 Regular"
英文标题: "Montserrat Bold"     # 或 "Helvetica Neue Bold"
英文正文: "Montserrat Regular"  # 或 "Helvetica Neue Regular"
数据字体: "DIN Alternate Bold"  # 或 "Arial Bold"
```
**适用**：企业汇报、经营分析、商务提案

### 组合2：科技极简

```yaml
中文标题: "苹方 Semibold"       # 或 "PingFang SC Semibold"
中文正文: "苹方 Regular"
英文标题: "SF Pro Display Bold"  # 或 "Inter Bold"
英文正文: "SF Pro Text Regular"  # 或 "Inter Regular"
数据字体: "SF Mono Bold"        # 或 "Roboto Mono Bold"
```
**适用**：技术方案、产品发布、互联网

### 组合3：优雅文艺

```yaml
中文标题: "思源宋体 Bold"
中文正文: "思源黑体 Light"
英文标题: "Playfair Display Bold"
英文正文: "Lato Regular"
数据字体: "DIN Alternate Bold"
```
**适用**：品牌展示、文化活动、高端发布

### 组合4：万能安全（系统自带）

```yaml
中文标题: "微软雅黑 Bold"
中文正文: "微软雅黑 Regular"
英文标题: "Arial Bold"
英文正文: "Arial Regular"
数据字体: "Arial Bold"
```
**适用**：任何场景（最大兼容性）

## 三、字号规范

### 标准字号体系

| 元素 | 字号 | 字重 | 行高 | 说明 |
|------|------|------|------|------|
| 主标题 | 40-48pt | Bold | 1.2 | 封面标题 |
| 章节标题 | 32-40pt | Bold | 1.2 | 过渡页标题 |
| 页面标题 | 28-36pt | Bold | 1.3 | 内容页标题 |
| 副标题 | 20-24pt | Medium | 1.3 | 二级标题 |
| 正文 | 16-20pt | Regular | 1.5 | 主要内容 |
| 注释 | 12-14pt | Light | 1.4 | 来源说明、辅助信息 |
| 数据大字 | 48-72pt | Bold | 1.0 | 核心数据展示 |

### 不同场景字号调整

| 场景 | 正文最小字号 | 原因 |
|------|-------------|------|
| 大会议室（>30人） | 20pt | 后排可读性 |
| 小会议室（<15人） | 16pt | 近距离阅读 |
| 1对1电脑展示 | 14pt | 近距离 |
| 屏幕共享/视频会议 | 18pt | 分辨率损失 |
| 打印阅读 | 10-12pt | 纸质阅读 |

## 四、排版黄金法则

### 4.1 对齐系统

**三种对齐方式**：

| 方式 | 适用场景 | 注意事项 |
|------|----------|----------|
| **左对齐** | 正文、列表、大段文字 | 最安全，可读性最好 |
| **居中对齐** | 标题、封面、少量文字 | 不超过3行 |
| **右对齐** | 数字、日期、辅助信息 | 仅用于特殊场景 |

**对齐参考线设置**：
```
┌──2cm──┬────────────────────────┬──2cm──┐
│       │                        │       │
│  2cm  │    内容安全区域          │  2cm  │
│       │                        │       │
│       │  ┌──参考线──┐          │       │
│       │  │          │          │       │
│       │  │  内容区   │          │       │
│       │  │          │          │       │
│       │  └──────────┘          │       │
│       │                        │       │
└───────┴────────────────────────┴───────┘
```

**页边距建议**：左右各2cm，上下各1.5cm

### 4.2 间距系统

| 间距类型 | 推荐值 | 用途 |
|----------|--------|------|
| 标题与正文 | 16-24px | 标题下方留空 |
| 段落间距 | 12-16px | 段与段之间 |
| 行间距 | 1.5倍 | 正文行距 |
| 列间距 | 24-40px | 多列布局 |
| 元素间距 | 16-24px | 卡片/图标间 |
| 页边距 | 2cm | 四周安全距离 |

### 4.3 文字排版禁忌

| 禁忌 | 正确做法 |
|------|----------|
| 大段文字不分段 | 每段不超过3-4行 |
| 行太长（超过20字/行） | 控制每行15字以内 |
| 字间距过紧或过松 | 保持默认或微调 |
| 全部文字同一大小 | 建立明确的字号层级 |
| 深色背景浅色小字 | 浅色背景深色文字可读性最好 |
| 文字贴边 | 留足页边距 |

## 五、文字效果技巧

### 5.1 强调文字的方法

| 方法 | 操作 | 视觉效果 |
|------|------|----------|
| **加粗** | 关键词Bold | 最常用，简单有效 |
| **变色** | 关键词用强调色 | 视觉跳跃，引导注意 |
| **放大** | 关键数据大字号 | 数据页常用 |
| **底色块** | 关键词加背景色块 | 标签式强调 |
| **下划线** | 关键词加下划线 | 简洁的强调 |

### 5.2 数据大字排版

```
┌──────────────────────────────┐
│                              │
│         ¥ 5.2 亿             │  ← 72pt Bold 强调色
│        Q3营业收入             │  ← 18pt Regular 灰色
│       同比增长 35% ↑          │  ← 24pt Bold 绿色
│                              │
└──────────────────────────────┘
```

### 5.3 列表排版

**方法一：图标列表**
```

```

**方法二：编号列表**
```

    补充说明文字

    补充说明文字

    补充说明文字
```

## 六、字体兼容性指南

### 安全字体清单（跨平台可用）

| 平台 | 安全中文字体 | 安全英文字体 |
|------|-------------|-------------|
| Windows + Mac | 微软雅黑 | Arial, Helvetica |
| Windows | 等线, 黑体 | Segoe UI, Calibri |
| Mac | 苹方, 华文黑体 | SF Pro, Helvetica Neue |
| 通用 Web | 思源黑体(Noto Sans SC) | Inter, Roboto |

### 字体嵌入建议

1. **优先使用系统字体** - 最大兼容性
2. **必须使用特殊字体时** - 在PPT中嵌入字体
3. **发送前转PDF** - 作为保底方案
4. **告知收件人字体需求** - 附带字体文件

## 七、排版自检清单

- [ ] 全篇字体不超过3种
- [ ] 标题和正文有明确的字号层级
- [ ] 正文字号不小于16pt
- [ ] 所有文字左对齐或居中对齐，不混用
- [ ] 行间距设置为1.5倍
- [ ] 页边距统一（左右2cm，上下1.5cm）
- [ ] 没有单独一行的孤字
- [ ] 中英文之间有半角空格
- [ ] 数字使用等宽字体
- [ ] 浅色背景上使用深色文字
