# PPT设计质量检查指南

> 本文档描述 AI 如何检查和诊断 PPT 的视觉设计质量

---

## 使用方式

当用户触发 `ppt-beautiful check` 命令时，AI 按以下流程进行设计质量检查。

---

## 检查维度

### 维度1：配色检查

| 检查项 | 合格标准 | 常见问题 |
|--------|----------|----------|
| 颜色数量 | 主要颜色≤4种 | 彩虹PPT |
| 色彩一致 | 全篇统一色板 | 每页颜色不同 |
| 对比度 | 文字背景对比≥4.5:1 | 浅色文字浅色背景 |
| 语义正确 | 正向绿色/负向红色 | 颜色含义混乱 |
| 饱和度 | 辅色饱和度适中 | 全部高饱和 |

**评分标准**：
- 5分：统一色板，对比清晰，语义正确
- 3分：基本统一，个别页面有偏差
- 1分：颜色杂乱，对比不足

### 维度2：字体检查

| 检查项 | 合格标准 | 常见问题 |
|--------|----------|----------|
| 字体数量 | ≤3种 | 多种字体混用 |
| 字号层级 | 标题>正文>注释 | 字号不分层级 |
| 最小字号 | ≥14pt（正文≥16pt） | 小字看不清 |
| 行间距 | 1.3-1.8倍 | 过密或过松 |
| 对齐方式 | 统一（左对齐或居中） | 混合对齐 |

**评分标准**：
- 5分：字体统一，层级清晰，间距舒适
- 3分：基本规范，个别页面不一致
- 1分：字体混乱，层级不清

### 维度3：版式检查

| 检查项 | 合格标准 | 常见问题 |
|--------|----------|----------|
| 对齐精度 | 所有元素有对齐参考 | 元素随意摆放 |
| 留白比例 | 页面填充≤70% | 内容挤满页面 |
| 间距统一 | 同类间距相等 | 间距不一致 |
| 页边距 | 四周留足安全距离 | 文字贴边 |
| 版式一致 | 同类页面使用同一版式 | 每页版式不同 |

**评分标准**：
- 5分：对齐精确，留白充足，一致性强
- 3分：大体对齐，留白适中
- 1分：杂乱无章，元素随意

### 维度4：图表检查

| 检查项 | 合格标准 | 常见问题 |
|--------|----------|----------|
| 图表类型 | 与数据关系匹配 | 类型选择不当 |
| 简洁度 | 去除多余装饰 | 3D效果、过多网格线 |
| 数据标注 | 关键数据有标注 | 只有图形无数据 |
| 标题质量 | 标题是结论 | 标题是描述 |
| 配色统一 | 与PPT整体风格一致 | 使用Excel默认色 |

### 维度5：整体一致性

| 检查项 | 合格标准 | 常见问题 |
|--------|----------|----------|
| 风格统一 | 全篇视觉风格一致 | 前后风格割裂 |
| 母版使用 | 使用了统一母版 | 无母版，每页独立 |
| 图标风格 | 统一线性或填充 | 线性+填充混搭 |
| 图片质量 | 高清无水印 | 模糊/有水印/拉伸 |
| 动画合理 | 简洁有目的 | 花哨或无动画 |

## 快速检查清单

用于快速自查，无需 AI 诊断：

### 10秒快检

- [ ] 快速翻页，风格是否统一？
- [ ] 颜色是否在3-4种以内？
- [ ] 字体是否不超过3种？

### 30秒细检

- [ ] 每页标题是否对齐？
- [ ] 留白是否充足？
- [ ] 图表是否简洁？
- [ ] 文字是否够大？

### 3分钟详检

- [ ] 配色方案是否统一？
- [ ] 字号层级是否清晰？
- [ ] 间距是否一致？
- [ ] 图标风格是否统一？
- [ ] 图片质量是否合格？
- [ ] 对齐是否精确？
