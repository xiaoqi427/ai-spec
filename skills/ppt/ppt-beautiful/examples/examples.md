# 使用示例

> PPT视觉设计大师（ppt-beautiful）技能的完整使用示例

---

## 示例1：完整视觉设计方案

### 用户输入

```bash
/skill ppt-beautiful design --topic "2026年度财务共享中心工作总结" --style "商务蓝" --pages 20
```

### AI 输出

```markdown
# PPT视觉设计方案：《2026年度财务共享中心工作总结》

## 视觉风格定位

| 要素 | 方案 |
|------|------|
| 风格 | 商务蓝 - 专业稳重 |
| 情感 | 专业、可信、成果感 |
| 比例 | 16:9 宽屏 |

## 配色方案

| 角色 | 色值 | 用途 |
|------|------|------|
| 主色 | #1A365D | 标题、装饰条、过渡页背景 |
| 辅色 | #3182CE | 次要标题、图表、图标 |
| 强调色 | #DD6B20 | 关键数据、CTA、重点标记 |
| 背景色 | #FFFFFF | 内容页背景 |
| 文字主色 | #2D3748 | 正文文字 |
| 文字副色 | #A0AEC0 | 注释、辅助文字 |
| 正向色 | #38A169 | 达标、增长 |
| 负向色 | #E53E3E | 未达标、下降 |

## 字体方案

| 用途 | 字体 | 字号 | 字重 |
|------|------|------|------|
| 主标题 | 思源黑体 | 40-48pt | Bold |
| 章节标题 | 思源黑体 | 32-36pt | Bold |
| 页面标题 | 思源黑体 | 28-32pt | Bold |
| 正文 | 思源黑体 | 18-20pt | Regular |
| 数据大字 | DIN Alternate | 48-72pt | Bold |
| 注释 | 思源黑体 | 12-14pt | Light |

## 版式规范

| 规范项 | 数值 |
|--------|------|
| 页边距 | 左右2cm，上下1.5cm |
| 标题位置 | 距顶部1.5cm，左对齐 |
| 标题下装饰线 | 宽80px，高3px，主色 |
| 内容区域 | 距标题20px，四周2cm安全距离 |
| 列间距 | 30px |
| 行间距 | 1.5倍 |

## 装饰元素

| 元素 | 规范 |
|------|------|
| 图标 | 线性风格，线宽2px，主色/辅色 |
| 装饰线 | 左侧竖线3px宽，主色 |
| 页码 | 右下角，14pt，灰色 |
| Logo | 右上角，固定大小 |

## 逐页视觉方案

### 第1页：封面 - 色块分割型
- 左40%：主色色块，白色大标题
- 右60%：白色背景，副标题+汇报信息
- Logo在右上角

### 第2页：目录 - 数字导航型
- 左对齐，4个章节
- 大数字（48pt 浅灰色）+ 章节名（24pt 深色）
- 当前章节用主色高亮

### 第3页：过渡页 - 大数字型
- 大数字"01"（120pt 浅色15%透明度）
- 章节标题（36pt Bold 深色）

### 第4-20页：内容页
- 统一版式，标题左上角
- 根据内容选择：
  - 数据页 → 四宫格KPI看板
  - 文字页 → 左右分割或三栏并列
  - 图表页 → 图表占页面60%
```

## 示例2：单页美化

### 用户输入

```bash
/skill ppt-beautiful page --type "数据页" --content "展示四个核心KPI：营收5.2亿、利润率18%、客户数156家、满意度98%"
```

### AI 输出

```markdown
# 数据页设计方案

## 推荐版式：四宫格KPI看板

```
┌──────────────────────────────┐
│  四大核心指标全面达标          │  ← 标题：观点式
│  ─────                       │  ← 装饰线
│                              │
│  ┌────────────┐ ┌────────────┐
│  │  ¥ 5.2 亿  │ │   18 %    │  ← 72pt Bold 主色
│  │  营业收入   │ │   利润率   │  ← 14pt Regular 灰色
│  │  ↑35%      │ │   ↑2%     │  ← 18pt Bold 绿色
│  │  同比增长   │ │   环比提升  │  ← 12pt Light 灰色
│  └────────────┘ └────────────┘
│  ┌────────────┐ ┌────────────┐
│  │  156 家    │ │   98 %    │
│  │  客户总数   │ │   满意度   │
│  │  ↑28%      │ │   ↑3%     │
│  │  净增加     │ │   连续提升  │
│  └────────────┘ └────────────┘
│                              │
└──────────────────────────────┘
```

## 设计要点

| 要素 | 规范 |
|------|------|
| 数据字号 | 48-72pt，DIN Alternate Bold |
| 标签字号 | 14pt，Regular，灰色 |
| 趋势字号 | 18pt，Bold，绿色/红色 |
| 卡片间距 | 24px |
| 卡片圆角 | 8px |
| 卡片背景 | #F7FAFC（极浅灰） |
| 正向箭头 | ↑ #38A169（绿色） |
| 负向箭头 | ↓ #E53E3E（红色） |
```

## 示例3：配色方案推荐

### 用户输入

```bash
/skill ppt-beautiful color --scene "科技产品发布" --mood "创新前沿"
```

### AI 输出

```markdown
# 配色方案推荐：科技产品发布

## 推荐方案：科技蓝紫

| 角色 | 色值 | 预览 | 用途 |
|------|------|------|------|
| 主色 | #2D1B69 | ████ | 深紫，标题和主视觉 |
| 辅色 | #553C9A | ████ | 中紫，渐变过渡 |
| 强调色 | #38B2AC | ████ | 青绿，点缀和CTA |
| 背景色 | #1A1A2E | ████ | 深色背景（深色模式） |
| 文字主色 | #E2E8F0 | ████ | 浅色文字 |
| 文字副色 | #A0AEC0 | ████ | 灰色辅助文字 |

## 应用示意

- **封面**：深色背景 + 渐变主色 + 白色大标题
- **内容页**：白色背景 + 紫色标题 + 青绿强调
- **数据页**：深色背景 + 青绿数据 + 白色标签
- **过渡页**：渐变背景（#2D1B69→#553C9A）+ 白色标题

## 备选方案

### 备选1：极简黑白

| 主色 #000000 | 辅色 #4A5568 | 强调色 #667EEA |

### 备选2：深空蓝

| 主色 #0D1B2A | 辅色 #1B2838 | 强调色 #00B4D8 |
```

## 示例4：设计诊断

### 用户输入

```bash
/skill ppt-beautiful check --file "这份PPT用了5种颜色，有的页面用宋体有的用黑体，图表直接从Excel复制的"
```

### AI 输出

```markdown
# PPT视觉设计诊断报告

## 总体评分：⭐⭐☆☆☆ (2/5)

### 各维度得分

| 维度 | 得分 | 状态 | 主要问题 |
|------|------|------|----------|
| 配色 | 2/5 | ❌ | 5种颜色过多，缺乏统一色板 |
| 字体 | 1/5 | ❌ | 宋体+黑体混用，风格冲突 |
| 版式 | 3/5 | ⚠️ | 需要确认对齐和留白 |
| 图表 | 1/5 | ❌ | Excel默认图表，需要全面美化 |
| 一致性 | 2/5 | ❌ | 字体和配色不统一 |

### 优化建议

#### 🔴 高优先级

1. **统一字体** → 全篇改用"思源黑体"
   - 标题：思源黑体 Bold 32pt
   - 正文：思源黑体 Regular 18pt
   - 宋体全部替换

2. **统一配色** → 从5种减至3种
   - 推荐：商务蓝色板（#1A365D + #3182CE + #DD6B20）
   - 删除多余颜色

3. **图表美化** → 重新制作图表
   - 去掉3D效果和多余网格线
   - 统一图表配色
   - 标题改为结论式

#### 🟡 中优先级

4. **版式规范化** → 使用统一母版
5. **间距统一** → 检查并调整所有间距
```
