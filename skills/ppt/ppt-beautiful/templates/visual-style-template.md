# 各类页面视觉设计模板

> 来源：《PPT，要你好看》 - 邵云蛟
> 为不同类型的PPT页面提供标准化的视觉设计方案

---

## 一、封面页设计

### 风格1：大图背景型

```
┌──────────────────────────────┐
│ ██████████████████████████████│
│ ██████████████████████████████│  ← 全幅高清大图 + 渐变蒙版
│ █████┌──────────────┐████████│
│ █████│  主标题       │████████│  ← 白色大字
│ █████│  副标题       │████████│  ← 白色小字
│ █████└──────────────┘████████│
│ ██████████████████████████████│
│ ████████ 汇报人 | 日期 ██████│
│ ██████████████████████████████│
└──────────────────────────────┘
```

**设计要点**：
- 图片选择：与主题相关的高清大图
- 蒙版处理：30-60%透明度的深色渐变蒙版
- 文字处理：白色/浅色文字，加轻微阴影增加可读性

### 风格2：色块分割型

```
┌──────────┬───────────────────┐
│          │                   │
│  主色块   │    浅色区域        │
│          │                   │
│  主标题   │    副标题          │
│  (白色)   │    汇报人/日期     │
│          │                   │
│  [Logo]  │    [装饰图形]      │
│          │                   │
└──────────┴───────────────────┘
```

**设计要点**：
- 左右分割比例：40:60 或 35:65
- 主色块使用主色，浅色区域使用背景色
- 装饰图形可选：线条、几何、图标

### 风格3：极简居中型

```
┌──────────────────────────────┐
│                              │
│                              │
│          [Logo]              │
│                              │
│     ━━━━━━━━━━━━━━━━━━━      │
│       主标题（大字）          │
│       副标题（小字）          │
│     ━━━━━━━━━━━━━━━━━━━      │
│                              │
│      汇报人 | 日期           │
│                              │
└──────────────────────────────┘
```

**设计要点**：
- 大量留白，突显简约气质
- 标题上下各一条细线装饰
- 适合极简风格

## 二、目录页设计

### 风格1：数字导航型

```
┌──────────────────────────────┐
│  CONTENTS                    │
│                              │
│  01 ─── 章节名称一            │
│  02 ─── 章节名称二            │
│  03 ─── 章节名称三            │
│  04 ─── 章节名称四            │
│                              │
└──────────────────────────────┘
```

### 风格2：卡片式

```
┌──────────────────────────────┐
│  目录                        │
│                              │
│  ┌──────┐ ┌──────┐          │
│  │  01  │ │  02  │          │
│  │章节一 │ │章节二 │          │
│  └──────┘ └──────┘          │
│  ┌──────┐ ┌──────┐          │
│  │  03  │ │  04  │          │
│  │章节三 │ │章节四 │          │
│  └──────┘ └──────┘          │
└──────────────────────────────┘
```

### 风格3：时间轴型

```
┌──────────────────────────────┐
│  目录                        │
│                              │
│  01 ──── 02 ──── 03 ──── 04 │
│  │       │       │       │  │
│  章节一  章节二  章节三  章节四│
│                              │
└──────────────────────────────┘
```

## 三、过渡页设计

### 风格1：大数字型

```
┌──────────────────────────────┐
│                              │
│                              │
│       01                     │  ← 超大数字（120pt 浅色）
│       ─────────              │
│       章节标题               │  ← 中等字号（36pt）
│       简短描述               │  ← 小字号（16pt 灰色）
│                              │
│                              │
└──────────────────────────────┘
```

### 风格2：色块全屏型

```
┌──────────────────────────────┐
│ ██████████████████████████████│
│ ██████████████████████████████│
│ ██████████████████████████████│  ← 全屏主色背景
│ ████████ 章节标题 ████████████│  ← 白色大字居中
│ ██████████████████████████████│
│ ██████████████████████████████│
│ ██████████████████████████████│
└──────────────────────────────┘
```

```
┌──────────────────────────────┐
│  页面标题                     │
│                              │
│  ┌──────────┐  ┌──────────┐ │
│  │          │  │ 要点一     │ │
```

## 五、人物介绍页

```
┌──────────────────────────────┐
│  核心团队                     │
│                              │
│  ┌────┐  ┌────┐  ┌────┐    │
│  │    │  │    │  │    │    │
│  │头像│  │头像│  │头像│    │
│  │    │  │    │  │    │    │
│  └────┘  └────┘  └────┘    │
│  姓名     姓名     姓名     │
│  职位     职位     职位     │
│  简介     简介     简介     │
│                              │
└──────────────────────────────┘
```

**设计要点**：
- 头像统一尺寸，圆形或圆角矩形
- 头像风格一致（照片/插画）
- 简介不超过2行
