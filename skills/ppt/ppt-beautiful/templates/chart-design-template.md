# 图表设计模板

> 来源：《PPT，要你好看》 - 邵云蛟
> 核心理念：图表是数据的视觉语言，简洁有力的图表胜过千言万语

---

## 一、图表选择矩阵

### 根据数据关系选图表

| 你想展示什么 | 推荐图表 | 变体 |
|-------------|----------|------|
| 趋势变化 | 折线图 | 面积图、阶梯图 |
| 大小比较 | 柱状图 | 条形图、瀑布图 |
| 占比构成 | 环形图 | 饼图、堆叠柱状图 |
| 分布情况 | 散点图 | 热力图、箱线图 |
| 流程关系 | 流程图 | 桑基图、泳道图 |
| 层级关系 | 树状图 | 矩形树图、组织架构 |
| 地理分布 | 地图 | 热力地图、气泡地图 |
| 相关性 | 散点图 | 气泡图 |

## 二、图表美化原则

### Before → After 改造

#### 柱状图美化

**Before（默认Excel风格）**：
- 有网格线
- 有边框
- 3D效果
- 图例在下方
- 颜色随机

**After（PPT风格）**：
- 去掉网格线
- 去掉边框
- 扁平化
- 直接在柱状上标注
- 统一配色，重点柱状用强调色

```
Before:                    After:
┌─┬─┬─┬─┬─┐              
│ │█│ │█│ │              ┌─┐ ┌─┐ ┌─┐ ┌─┐ ┌─┐
│ │█│█│█│█│              │ │ │ │ │█│ │ │ │ │
│█│█│█│█│█│              │ │ │ │ │█│ │ │ │ │
│█│█│█│█│█│              │ │ │ │ │█│ │ │ │ │
└─┴─┴─┴─┴─┘              └─┘ └─┘ └─┘ └─┘ └─┘
(网格线密布)               (干净，重点突出)
```

### 图表简化清单

去掉以下元素：
- [ ] 网格线（或保留极淡的水平参考线）
- [ ] 图表边框
- [ ] 3D效果
- [ ] 阴影效果
- [ ] 数据表格（直接在图表上标注）
- [ ] 不必要的图例（直接标注在数据旁）
- [ ] Y轴刻度线（保留关键刻度即可）

## 三、各类图表设计指南

### 1. 柱状图/条形图

**设计规范**：
```yaml
柱宽: 柱间距的1.5-2倍
圆角: 建议4-8px圆角
颜色: 普通柱=辅色，重点柱=强调色
数据标签: 柱顶显示，字号14-16pt
```

**突出技巧**：
- 只给重点数据上色，其余用灰色
- 在重点柱上方添加数据标签
- 用箭头或标注指出关键变化

### 2. 折线图/面积图

**设计规范**：
```yaml
线条粗细: 2-3pt
数据点: 关键节点显示圆点
颜色: 主系列用主色，对比系列用辅色
填充: 面积图用渐变填充（上深下浅，20-60%透明度）
```

**突出技巧**：
- 在关键转折点添加标注气泡
- 用垂直虚线标记重要时间节点
- 添加趋势线或目标参考线

### 3. 环形图/饼图

**设计规范**：
```yaml
推荐: 环形图优于饼图（中间可放数据）
扇区数: 不超过5-6个，其余归入"其他"
环宽: 环形图内径为外径的50-60%
颜色: 主扇区=主色/强调色，其余=灰色系
```

**中心区域利用**：
```
┌─────────────────┐
│    ┌─────────┐  │
│    │  ¥5.2亿 │  │  ← 环形图中心放核心数据
│    │  总营收  │  │
│    └─────────┘  │
│  ████████████   │  ← 环形图环
│  █████  █████   │
│  ████████████   │
└─────────────────┘
```

### 4. 数据看板（KPI卡片）

**单指标卡片**：
```
┌──────────────┐
│   ¥ 5.2亿    │  ← 72pt Bold 主色
│   营业收入    │  ← 14pt Regular 灰色
│   ↑ 35%      │  ← 18pt Bold 绿色
│   同比增长    │  ← 12pt Light 灰色
└──────────────┘
```

**四指标看板**：
```
┌──────────┬──────────┐
│  ¥5.2亿  │  18%     │
│  营收     │  利润率   │
│  ↑35%    │  ↑2%     │
├──────────┼──────────┤
│  12家    │  98%     │
│  新客户   │  满意度   │
│  ↑50%    │  ↑3%     │
└──────────┴──────────┘
```

### 5. 对比图表

**双柱对比**：
```

指标A    ████   ██████  120%
```

### 6. 流程图/时间轴

**水平流程**：
```
  ┌───┐     ┌───┐     ┌───┐     ┌───┐
  │ 1 │────→│ 2 │────→│ 3 │────→│ 4 │
  └───┘     └───┘     └───┘     └───┘
  步骤一    步骤二    步骤三    步骤四
  说明      说明      说明      说明
```

**垂直时间轴**：
```
  2024.Q1  ●── 里程碑1：xxx
           │
  2024.Q2  ●── 里程碑2：xxx
           │
  2024.Q3  ●── 里程碑3：xxx  ← 当前
           │
  2024.Q4  ○── 里程碑4：xxx  (待完成)
```

## 四、图表配色规范

### 单系列配色

```
重点数据: 强调色（如 #DD6B20）
普通数据: 辅色（如 #A0AEC0）
```

### 多系列配色

```
系列1: 主色
系列2: 辅色
系列3: 强调色
系列4: 主色浅色版
系列5: 辅色浅色版
```

### 语义色彩

| 含义 | 推荐色 | 色值 |
|------|--------|------|
| 正向/增长/达标 | 绿色 | #38A169 |
| 负向/下降/未达标 | 红色 | #E53E3E |
| 警告/注意 | 黄色 | #D69E2E |
| 中性/基准 | 灰色 | #A0AEC0 |
| 信息/参考 | 蓝色 | #3182CE |

## 五、图表标题写法

| 差的标题 | 好的标题 |
|----------|----------|
| "2024年销售额" | "2024年销售额创历史新高，达¥5.2亿" |
| "市场份额" | "华东市场份额升至28%，位居第一" |
| "成本分析" | "人力成本占比超60%，优化空间巨大" |
| "用户增长" | "月活用户突破500万，同比增长120%" |

**标题公式**：`结论/观点 + 关键数据`

## 六、图表设计自检清单

- [ ] 图表类型与数据关系匹配
- [ ] 去除了不必要的装饰元素
- [ ] 重点数据用强调色突出
- [ ] 标题是结论而非描述
- [ ] 数据标签清晰可读
- [ ] 配色与PPT整体风格一致
- [ ] 3秒内能看懂核心信息
