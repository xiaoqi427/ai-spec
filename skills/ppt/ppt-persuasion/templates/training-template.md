# 培训教学PPT模板

> 基于《说服力》七步法设计的培训教学场景模板

---

## 模板概况

| 属性 | 说明 |
|------|------|
| 适用场景 | 内部培训/技术分享/知识讲座/workshop |
| 推荐页数 | 30-60页 |
| 演示时长 | 45-120分钟 |
| 核心听众 | 学员/团队成员/新人 |
| 风格建议 | 清晰易读、图文并茂、互动性强 |

---

## 结构框架

```
页码     | 类型     | 内容
---------|----------|------------------
第1页    | 封面     | 课程标题 + 讲师 + 日期
第2页    | 引入     | 一个问题或场景引发思考
第3页    | 目录     | 学习大纲（4-6个模块）
第4页    | 目标     | 学完后能做什么
第5页    | 过渡页   | 模块一标题
第6-12页 | 内容页   | 知识点 + 案例 + 练习
第13页   | 互动页   | 小测/讨论/动手练习
          ... (重复模块结构)
倒数3页  | 总结     | 知识图谱/要点回顾
倒数2页  | 行动     | 课后作业/实践计划
最后1页  | 致谢     | 反馈收集 + 联系方式
```

## 逐页设计指南

### 第1页：封面

```
┌──────────────────────────────┐
│                              │
│    [课程图标/插画]            │
│                              │
│   《课程标题》                │
│    副标题：适用人群/级别       │
│                              │
│   讲师：XXX | 时间：XXXX     │
│   时长：XX分钟               │
│                              │
└──────────────────────────────┘
```

### 第2页：引入（制造好奇）

**技巧**：用一个问题或场景开场

| 引入方式 | 示例 |
|----------|------|
| 提问式 | "你知道95%的PPT为什么没人看吗？" |
| 故事式 | "上周，小张的汇报被老板打回来3次..." |
| 数据式 | "据统计，平均每人一年要看300+份PPT" |
| 痛点式 | "你是否也遇到过这些问题？" |

### 第3页：目录（学习路线图）

```
┌──────────────────────────────┐
│  学习路线图                   │
│                              │
│  ① 基础概念 ──→ ② 核心方法   │
│       │               │      │
│       ↓               ↓      │
│  ③ 实战演练 ──→ ④ 进阶技巧   │
│       │               │      │
│       ↓               ↓      │
│  ⑤ 案例分析 ──→ ⑥ 总结行动   │
│                              │
└──────────────────────────────┘
```

### 第4页：学习目标

```
学完本课程，你将能够：

✅ 目标1：掌握XXX的核心概念
✅ 目标2：学会使用XXX方法
✅ 目标3：独立完成XXX任务
✅ 目标4：解决XXX常见问题
```

### 内容页设计原则

**每个知识点遵循 "讲→示→练" 三步**：

| 步骤 | 内容 | 页数 |
|------|------|------|
| 讲（Teach） | 概念讲解 + 原理说明 | 1-2页 |
| 示（Show） | 案例展示 + 对比分析 | 1-2页 |
| 练（Practice） | 互动练习 + 即时反馈 | 1页 |

**知识点页面版式**：
```
┌──────────────────────────────┐
│  观点式标题                   │
│                              │
│  ┌────────┐  ┌────────────┐ │
│  │        │  │ 要点1       │ │
│  │  图示   │  │ 要点2       │ │
│  │  案例   │  │ 要点3       │ │
│  │  图表   │  │            │ │
│  └────────┘  └────────────┘ │
│                              │
│  💡 关键提示：一句话总结      │
│                              │
└──────────────────────────────┘
```

### 互动页类型

| 类型 | 说明 | 时机 |
|------|------|------|
| 小测验 | 2-3道选择题检验理解 | 每个模块后 |
| 讨论题 | 开放式问题引发思考 | 关键概念后 |
| 动手练习 | 实操任务 | 技能类内容后 |
| 案例分析 | 给出场景，学员分析 | 综合应用时 |
| 小组活动 | 分组讨论/竞赛 | 中间或结尾 |

│  ┌──┴──┐┌─┴──┐┌─┴──┐        │

## 培训PPT特别注意

1. **字号不小于24pt**（教室后排可读性）
2. **每页停留2-3分钟**（控制节奏）
3. **每15分钟设置一次互动**（保持注意力）
4. **代码/公式用等宽字体**（清晰可辨）
5. **关键概念用色块高亮**（视觉锚点）
