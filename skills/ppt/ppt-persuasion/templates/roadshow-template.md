# 商业路演PPT模板

> 基于《说服力》七步法设计的商业路演/投资路演场景模板

---

## 模板概况

| 属性 | 说明 |
|------|------|
| 适用场景 | 融资路演/商业计划/产品发布 |
| 推荐页数 | 15-20页 |
| 演示时长 | 15-20分钟 |
| 核心听众 | 投资人/合作伙伴/客户高管 |
| 风格建议 | 精美大气、故事化、数据有力 |

---

## 结构框架（SCQA故事线）

```
页码    | 类型     | SCQA    | 内容
--------|----------|---------|------------------
第1页   | 封面     |         | 品牌名 + 一句话定位
第2页   | 痛点     | S+C     | 市场痛点/行业问题
第3页   | 机会     | Q       | 市场机会有多大
第4页   | 方案     | A       | 我们的解决方案
第5页   | 产品     | A       | 产品核心功能/服务
第6-7页 | 优势     | A       | 竞争优势/差异化
第8页   | 模式     | A       | 商业模式/盈利模式
第9页   | 市场     |         | 市场规模/TAM-SAM-SOM
第10页  | 数据     |         | 核心运营数据/里程碑
第11页  | 案例     |         | 客户案例/成功故事
第12页  | 团队     |         | 核心团队介绍
第13页  | 竞对     |         | 竞争格局分析
第14页  | 规划     |         | 未来发展规划
第15页  | 财务     |         | 财务预测/融资需求
第16页  | 结尾     |         | 愿景 + 联系方式
```

---

## 逐页设计指南

### 第1页：封面（品牌定位）

```
┌──────────────────────────────┐
│                              │
│        [品牌Logo]            │
│                              │
│    一句话描述你的项目         │
│    "让XX变得更简单"           │
│                              │
│    路演人 | 日期 | 轮次       │
│                              │
└──────────────────────────────┘
```

**要求**：30秒内让投资人理解你做什么

### 第2页：痛点（制造共鸣）

**模式**：讲一个故事或描述一个场景
- 用户面临什么问题？
- 这个问题有多严重？（量化）
- 现有解决方案为什么不够好？

### 第3页：市场机会

```
┌──────────────────────────────┐
│  TAM-SAM-SOM 市场模型        │
│                              │
│     ┌─────────────────┐      │
│     │   TAM: ¥XXX亿   │      │
│     │  ┌───────────┐  │      │
│     │  │ SAM: ¥XX亿│  │      │
│     │  │ ┌───────┐ │  │      │
│     │  │ │SOM:¥X亿│ │  │      │
│     │  │ └───────┘ │  │      │
│     │  └───────────┘  │      │
│     └─────────────────┘      │
│                              │
│  数据来源：XX研究报告         │
└──────────────────────────────┘
```

### 第4-5页：解决方案 + 产品

- 用一句话说清楚你的方案
- 产品截图或架构图
- 核心功能不超过3个

### 第6-7页：竞争优势

**推荐**：竞争矩阵对比图

| 维度 | 我们 | 竞对A | 竞对B |
|------|------|-------|-------|
| 功能 | ✅✅✅ | ✅✅ | ✅ |
| 价格 | ✅✅✅ | ✅ | ✅✅ |
| 体验 | ✅✅✅ | ✅✅ | ✅✅ |

### 第8页：商业模式

- 清晰展示盈利方式
- 收入来源和定价策略
- 用图示而非纯文字

### 第9-11页：数据与案例

**运营数据**（关键指标看板）：
- 用户数、收入、增长率、留存率
- 用折线图展示增长趋势

**客户案例**：
- 标杆客户Logo墙
- 1-2个具体成功案例

### 第12页：团队

```
┌──────────────────────────────┐
│  核心团队                     │
│                              │
│  ┌────┐  ┌────┐  ┌────┐     │
│  │头像│  │头像│  │头像│     │
│  │CEO │  │CTO │  │COO │     │
│  │简介│  │简介│  │简介│     │
│  └────┘  └────┘  └────┘     │
│                              │
│  关键词：XX年行业经验/XX背景  │
│                              │
└──────────────────────────────┘
```

### 第15页：融资需求

- 融资金额和轮次
- 资金用途分配（饼图）
- 未来12-18个月关键里程碑

### 第16页：结尾（愿景召唤）

```
┌──────────────────────────────┐
│                              │
│     "我们的愿景是..."        │
│                              │
│     [一句有感染力的话]        │
│                              │
│     联系人 | 邮箱 | 电话     │
│                              │
└──────────────────────────────┘
```

## 配色建议

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | #2D3748（深灰蓝） | 专业、可信 |
| 品牌色 | 使用品牌主色 | 品牌识别 |
| 强调色 | #F6AD55（金橙） | 重点数据 |
| 正向 | #48BB78（绿色） | 增长、优势 |
| 背景 | #FFFFFF 或 #F7FAFC | 干净、简洁 |
