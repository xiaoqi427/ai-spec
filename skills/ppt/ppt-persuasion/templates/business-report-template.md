# 工作汇报PPT模板

> 基于《说服力》七步法设计的工作汇报场景模板

---

## 模板概况

| 属性 | 说明 |
|------|------|
| 适用场景 | 周报/月报/季报/年度总结 |
| 推荐页数 | 15-25页 |
| 演示时长 | 15-30分钟 |
| 核心听众 | 上级领导/部门负责人 |
| 风格建议 | 商务简约、数据驱动 |

---

## 结构框架

```
页码    | 类型     | 内容
--------|----------|------------------
第1页   | 封面     | 标题 + 汇报人 + 日期
第2页   | 目录     | 章节导航（4个板块）
第3页   | 过渡页   | 第一章：工作回顾
第4-7页 | 内容页   | 关键成果 + 核心数据
第8页   | 过渡页   | 第二章：问题与挑战
第9-11页| 内容页   | 问题分析 + 根因 + 影响
第12页  | 过渡页   | 第三章：改进方案
第13-16页| 内容页  | 解决策略 + 行动计划
第17页  | 过渡页   | 第四章：下一步计划
第18-20页| 内容页  | 目标 + 里程碑 + 资源需求
第21页  | 总结页   | 核心要点回顾
第22页  | 致谢页   | Q&A + 联系方式
```

---

## 逐页设计指南

### 第1页：封面

```
┌──────────────────────────────┐
│                              │
│     [公司Logo]               │
│                              │
│   《主标题：核心结论式命名》    │
│    副标题：时间范围/部门       │
│                              │
│   汇报人：XXX  日期：XXXX     │
│                              │
└──────────────────────────────┘
```

**标题示例**：
- 差："Q3工作汇报"
- 好："Q3营收超额达成，利润率提升3个百分点"

### 第2页：目录

```
┌──────────────────────────────┐
│  目录 CONTENTS               │
│                              │
│  01 工作回顾 ─── 关键成果     │
│  02 问题分析 ─── 挑战与根因   │
│  03 改进方案 ─── 策略与行动   │
│  04 下步计划 ─── 目标与资源   │
│                              │
└──────────────────────────────┘
```

### 第3-7页：工作回顾

**核心要求**：用数据说话，突出成果

**第4页 - 核心数据看板**：
```
┌──────────────────────────────┐
│  Q3核心指标达成情况           │
│                              │
│  ┌────┐ ┌────┐ ┌────┐ ┌────┐│
│  │营收 │ │利润 │ │客户 │ │效率 ││
│  │120% │ │115%│ │108%│ │ 95%││
│  │ ↑   │ │ ↑  │ │ ↑  │ │ ↓  ││
│  └────┘ └────┘ └────┘ └────┘│
│                              │
│  [趋势折线图]                │
│                              │
└──────────────────────────────┘
```

**第5-7页 - 重点工作回顾**（每页一个重点）：
- 版式：左右结构（左文右图）
- 标题：观点式（"XX项目提前交付，客户满意度达98%"）
- 内容：3-4个关键要点 + 数据支撑

### 第8-11页：问题与挑战

**核心要求**：坦诚面对，分析根因

**第9页 - 问题清单**：
- 用红/黄/绿标记问题严重程度
- 每个问题一句话描述

**第10-11页 - 根因分析**：
- 使用鱼骨图或5Why分析法
- 区分内因和外因
- 量化影响程度

### 第12-16页：改进方案

**核心要求**：有具体措施和时间表

**第13页 - 策略概览**：
- 3-5个核心策略
- 每个策略一句话 + 预期效果

**第14-16页 - 行动计划**：
- 甘特图或时间轴展示
- 明确责任人和截止时间
- 标注所需资源和支持

### 第17-20页：下一步计划

**核心要求**：目标明确，资源请求具体

**第18页 - 下阶段目标**：
- SMART目标定义
- 对齐上级战略方向

**第19-20页 - 里程碑和资源**：
- 关键里程碑时间节点
- 所需资源和支持请求

### 第21页：总结

```
┌──────────────────────────────┐
│  核心要点                     │
│                              │
│  ✅ 成果1：[一句话]           │
│  ⚠️ 挑战1：[一句话]           │
│  🎯 计划1：[一句话]           │
│                              │
│  需要的支持：                 │
│  1. [具体支持请求]            │
│  2. [具体支持请求]            │
│                              │
└──────────────────────────────┘
```

## 配色建议

| 用途 | 色值 | 场景 |
|------|------|------|
| 主色 | #1A365D（深蓝） | 标题、强调 |
| 辅色 | #E2E8F0（浅灰蓝） | 背景、辅助 |
| 强调色 | #DD6B20（橙色） | 关键数据、重点 |
| 正向 | #38A169（绿色） | 达标、增长 |
| 负向 | #E53E3E（红色） | 未达标、下降 |
