# 芒格投资检查清单 (Munger Checklist)

> 使用场景: 框架A Step5 决策前必须逐项检查

## 检查清单（25项）

### 一、能力圈检查（5项）
- [ ] C01. 我能用一句话说清这家公司的商业模式吗?
- [ ] C02. 我理解这个行业的核心竞争要素吗?
- [ ] C03. 我知道这家公司的主要客户和盈利来源吗?
- [ ] C04. 我能列出至少3个这家公司可能失败的原因吗?
- [ ] C05. 如果把公司信息全部删除只留财务数据，我还能辨认出它吗?

### 二、护城河检查（5项）
- [ ] C06. 这家公司有明确的竞争优势吗?（品牌/网络/转换成本/规模/牌照）
- [ ] C07. 这个优势5年后还会存在吗?
- [ ] C08. 如果给竞争对手10亿资金，它能复制这家公司吗?
- [ ] C09. 这家公司的定价权强不强?（是否能涨价而不失去客户）
- [ ] C10. 行业新进入者的威胁大吗?

### 三、管理层检查（5项）
- [ ] C11. 管理层过往承诺的兑现率高吗?
- [ ] C12. 管理层是否与股东利益一致?（持股/薪酬结构）
- [ ] C13. 管理层在困难时期的表现如何?
- [ ] C14. 是否有不诚信的历史记录?
- [ ] C15. 管理层是否过度乐观?（对比预期与实际）

### 四、估值与风险检查（5项）
- [ ] C16. 当前价格是否提供了足够的安全边际?（≥20%）
- [ ] C17. 如果这笔投资亏损50%，我能承受吗?
- [ ] C18. 最坏情况下（反转思考），我的最大损失是多少?
- [ ] C19. 有没有更好的替代投资?（机会成本）
- [ ] C20. 这个价格是否反映了市场的过度乐观/悲观?

### 五、自我检查（5项）
- [ ] C21. 我的决策是否受到了情绪影响?（贪婪/恐惧）
- [ ] C22. 我是否只看到了支持自己观点的证据?（确认偏差）
- [ ] C23. 我的信息是否足够充分?
- [ ] C24. 我能清晰地向别人解释我的投资逻辑吗?（极度透明）
- [ ] C25. 这笔投资符合我的长期策略吗?

## 评分规则
- 每项通过得1分，总分25分
- ≥ 22分: 可以执行
- 18-21分: 需要补充研究后再决定
- < 18分: 不建议操作
