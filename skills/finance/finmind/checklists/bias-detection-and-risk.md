# 认知偏差自查清单 (Bias Detection Checklist)

> 使用场景: WORKFLOW 阶段3（强制环节，不可跳过）

## 自查清单（12项关键偏差）

### 即时扫描（每次决策前必查）

| 编号 | 偏差 | 自查问题 | 红旗信号 |
|------|------|---------|---------|
| B01 | **锚定效应** | 我是否被某个价格（买入价/历史高点）锚定了? | "它曾经到过XX元" |
| B02 | **确认偏差** | 我是否只看到了支持我观点的证据? | 只搜索利好新闻 |
| B03 | **损失厌恶** | 我是否因为怕亏钱而不愿止损? | "再等等可能会回来" |
| B04 | **过度自信** | 我对自己的判断是否过于确定? | "这次肯定没问题" |
| B05 | **羊群效应** | 我是否因为别人都在买而买? | "大家都说好" |
| B06 | **近因偏差** | 我是否过度关注最近的信息? | 仅因为最近涨了就看好 |

### 深度扫描（重大决策前加查）

| 编号 | 偏差 | 自查问题 | 红旗信号 |
|------|------|---------|---------|
| B07 | **沉没成本** | 我是否因为已经投入而不愿放弃? | "已经亏了这么多" |
| B08 | **处置效应** | 我是否急着卖盈利的，死扛亏损的? | 盈利马上走，亏损不动 |
| B09 | **过度交易** | 我最近交易是否过于频繁? | 每天都想操作 |
| B10 | **故事偏差** | 我是否被"好故事"吸引而忽略数据? | "这个概念太好了" |
| B11 | **权威偏差** | 我是否因为某个大V推荐才感兴趣? | "XX说的应该没错" |
| B12 | **控制幻觉** | 我是否觉得自己能预测市场走向? | "我判断明天会涨" |

## 使用方法

每项标注: ✅ 未检测到 / ⚠️ 轻微存在 / ❌ 明显存在

- 全部 ✅: 偏差检测通过
- 有 ⚠️: 通过但需注意，标注提醒
- 有 ❌: 需要暂停决策，冷静重新评估

# 风控合规检查清单 (Risk Control Checklist)

> 使用场景: 每次交易执行前

## 检查项

### 仓位检查
- [ ] R01. 本次操作后，单只个股仓位 ≤ 30%?
- [ ] R02. 本次操作后，同行业仓位 ≤ 40%?
- [ ] R03. 本次操作后，总持仓 ≤ 80%?
- [ ] R04. 本次操作后，现金保留 ≥ 20%?

### 止损检查
- [ ] R05. 是否已确定止损位?
- [ ] R06. 单笔最大亏损 ≤ 总资金2%?
- [ ] R07. 个股止损线 ≤ -8%?

### 计划检查
- [ ] R08. 是否有完整的交易计划（入场/止损/目标/仓位）?
- [ ] R09. 计划是否在冷静状态下制定的?（非盘中冲动）
- [ ] R10. 是否已通过芒格检查清单?

### 段永平三不原则
- [ ] R11. 不做空 ✅
- [ ] R12. 不借钱投资 ✅
- [ ] R13. 不做不懂的 ✅

## 评分规则
- 全部通过(13/13): 允许执行
- 10-12项通过: 需调整后执行
- < 10项通过: 否决执行
