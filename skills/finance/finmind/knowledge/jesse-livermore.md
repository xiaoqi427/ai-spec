# 利弗莫尔交易智慧 —《股票作手回忆录》

> **"华尔街没有新鲜事，因为投机像群山一样古老。"**

## 核心交易原则

### 趋势为王
- "股票永远沿着阻力最小的方向运动"
- 不要和市场争论，跟随趋势
- 上升趋势中做多，下降趋势中做空或空仓
- 趋势不明时，最好的操作是不操作

### 关键位理论（最重要的概念）
```
关键位 = 市场集体心理的临界点

类型:
  - 历史高点/低点
  - 整数关口（100/500/1000等）
  - 前期成交密集区
  - 跳空缺口

判断方法:
  放量突破关键位 → 趋势加速（利弗莫尔最重要的买入信号）
  缩量受阻关键位 → 可能反转
  在关键位反复争夺 → 等待方向明确

"股票在关键价位的表现，比任何其他信息都重要"
```

### 止损纪律（生存法则）
- "止损是保险费。不买保险的人，终将付出更大的代价"
- "永远不要移动止损位来扩大亏损"
- "止损让你保留资本，资本是你继续交易的基础"
- **铁律**: 在入场前就确定止损位，触发时无条件执行

### 耐心等待
- "赚大钱不是靠交易频繁，而是靠等待"
- "坐着不动赚的钱，比频繁交易赚的钱多得多"
- 等待正确的时机，不要为了交易而交易
- 趋势确立后，持有直到趋势结束

## 交易方法

### 试探性建仓
```
利弗莫尔的建仓法:

1. 先用小仓位试探（总资金的10-20%）
2. 如果方向正确且盈利 → 加仓
3. 如果方向错误且亏损 → 止损出局
4. 永远不要在亏损时加仓（摊低成本是陷阱）

原则:
  "让利润奔跑，截断亏损"
  "加仓只在盈利时，止损必须及时"
```

### 市场心理洞察
- 大众在顶部最乐观，在底部最悲观
- 当所有人都认为会涨的时候，往往是顶部
- 当所有人都认为会跌的时候，往往是底部
- "市场从不错，错的永远是人的看法"

### 利弗莫尔的交易规则
1. **顺势而为**: 永远不要逆势操作
2. **等待确认**: 不要抢在趋势确立之前入场
3. **及时止损**: 小亏损比大亏损好一万倍
4. **让利润奔跑**: 不要过早止盈
5. **不要预测**: 对市场保持敬畏
6. **控制情绪**: 恐惧和贪婪是最大的敌人
7. **休息也是操作**: 趋势不明时，空仓等待

## 失败教训（同样重要）

利弗莫尔多次破产的教训:
1. **过度自信** → 仓位过重
2. **违反纪律** → 没有及时止损
3. **听信他人** → 失去独立判断
4. **情绪失控** → 报复性交易
5. **不休息** → 在不该交易的时候交易
