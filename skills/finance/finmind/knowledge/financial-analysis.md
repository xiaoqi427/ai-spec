# 财务分析方法论

> 融合《财务报表分析与股票估值》+《从财务分析到经营分析》

## 三表分析体系

### 利润表（Income Statement）— 公司赚了多少钱
```
核心指标:
  营业收入 → 公司的"顶线"，反映市场需求
  毛利率 → 产品竞争力的直接体现
  营业利润率 → 扣除费用后的真实盈利能力
  净利率 → 最终落袋的比例
  
关注要点:
  1. 收入增长的质量（靠量还是靠价?）
  2. 毛利率趋势（上升=竞争力增强）
  3. 费用率变化（销售/管理/研发费用占比）
  4. 非经常性损益（扣非净利润更真实）
```

### 资产负债表（Balance Sheet）— 公司有什么、欠什么
```
核心指标:
  总资产 → 公司的"家底"
  净资产 → 属于股东的部分
  资产负债率 → 杠杆水平
  流动比率 → 短期偿债能力
  
关注要点:
  1. 资产质量（应收账款是否过多? 存货是否积压?）
  2. 负债结构（有息负债vs经营负债）
  3. 商誉规模（高商誉=潜在减值风险）
  4. 货币资金充裕度
```

### 现金流量表（Cash Flow Statement）— 公司的血液
```
三种现金流:
  经营现金流(OCF) → 主营业务产生的现金（最重要）
  投资现金流(ICF) → 扩张/收缩的信号
  筹资现金流(FCF) → 融资/还债的信号
  
核心判断:
  OCF/净利润 > 80% → 盈利质量好（真金白银）
  OCF/净利润 < 50% → 盈利质量差（纸上富贵，需警惕）
  
自由现金流 = 经营现金流 - 资本支出
  自由现金流 > 0 → 公司能自我造血
  自由现金流 < 0 → 依赖外部融资
```

## 关键财务指标速查

### 盈利能力
| 指标 | 计算公式 | 优秀 | 良好 | 警惕 |
|------|---------|------|------|------|
| ROE | 净利润/净资产 | >20% | 15-20% | <10% |
| ROIC | NOPAT/(净资产+有息负债) | >WACC×1.5 | >WACC | <WACC |
| 毛利率 | 毛利/营收 | >50% | 30-50% | <20% |
| 净利率 | 净利润/营收 | >20% | 10-20% | <5% |

### 成长能力
| 指标 | 计算公式 | 高速 | 稳健 | 停滞 |
|------|---------|------|------|------|
| 营收增速 | (本期-上期)/上期 | >30% | 10-30% | <5% |
| 净利增速 | 同上 | >30% | 10-30% | <5% |
| 自由现金流增速 | 同上 | >净利增速 | ≈净利增速 | <净利增速 |

### 安全性
| 指标 | 计算公式 | 安全 | 一般 | 危险 |
|------|---------|------|------|------|
| 资产负债率 | 总负债/总资产 | <40% | 40-60% | >70% |
| 流动比率 | 流动资产/流动负债 | >2.0 | 1.5-2.0 | <1.0 |
| 有息负债率 | 有息负债/总资产 | <20% | 20-40% | >50% |

## 杜邦分析拆解

```
ROE = 净利率 × 资产周转率 × 权益乘数(杠杆)

三种高ROE模式:
  
  模式一: 高净利率（最优）
    代表: 茅台、腾讯
    特点: 护城河带来定价权
    风险: 低
    
  模式二: 高周转率（良好）
    代表: 沃尔玛、Costco
    特点: 效率和管理能力
    风险: 中
    
  模式三: 高杠杆（需警惕）
    代表: 银行、地产
    特点: 借钱赚钱
    风险: 高
```

## 估值方法

### 绝对估值 — DCF
```
公司价值 = Σ (FCFt / (1+WACC)^t) + 终值/(1+WACC)^n

关键参数:
  FCF: 未来自由现金流预测
  WACC: 加权平均资本成本(通常8-12%)
  g: 永续增长率(通常2-3%)
  n: 预测年限(通常10年)
  
注意: DCF对假设非常敏感，结果是一个区间而非精确值
```

### 相对估值
| 方法 | 适用 | 判断标准 |
|------|------|---------|
| PE(市盈率) | 盈利稳定的公司 | 历史分位<25%=低估 |
| PB(市净率) | 重资产公司/银行 | PB<1可能低估 |
| PS(市销率) | 亏损但有收入的公司 | 同行比较 |
| PEG | 成长型公司 | PEG<1=合理 |
| EV/EBITDA | 资本密集型行业 | 同行比较 |

## 从财务分析到经营分析

```
财务分析只是起点，经营分析才是目的:

财务数字 → 为什么? → 经营本质

示例:
  毛利率下降 → 为什么? → 原材料涨价? 竞争加剧? 产品过时?
  应收账款增加 → 为什么? → 放宽信用? 客户质量下降? 行业惯例?
  存货增加 → 为什么? → 备货旺季? 滞销积压? 囤积原材料?
  
核心: 不要停留在数字本身，要追问数字背后的业务逻辑
```
