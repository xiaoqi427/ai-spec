# 数据分析精要 —《数据交易：数据解析精要》

> **"没有高质量的数据，就没有高质量的决策。"**

## 数据质量评估框架

### 六维度评估模型
| 维度 | 含义 | 投资数据评估 |
|------|------|-------------|
| **完整性** | 数据是否齐全，无缺失 | 财务数据是否完整覆盖所有季度 |
| **准确性** | 数据是否正确反映真实情况 | 数据源是否权威（交易所/官方披露） |
| **一致性** | 多源数据是否吻合 | 不同平台的同一指标是否一致 |
| **时效性** | 数据是否及时更新 | 行情是否实时/财报是否最新 |
| **唯一性** | 数据是否去重 | 同一条新闻是否被重复计入 |
| **有效性** | 数据是否符合业务规则 | PE为负数时是否正确处理 |

### 数据清洗标准流程
```
原始数据 → 去重 → 缺失值处理 → 异常值检测 → 标准化 → 可用数据

1. 去重: 相同时间戳的重复数据
2. 缺失值: 
   - 行情数据: 用前一交易日填充(forward fill)
   - 财务数据: 标注"缺失"，不做插值
3. 异常值:
   - 价格异常(如突然涨跌99%): 标记待人工确认
   - 财务异常(如负数营收): 检查数据源
4. 标准化:
   - 统一时间格式(ISO 8601)
   - 统一货币单位(人民币/美元)
   - 统一数值精度(价格2位/比率4位)
```

## 另类数据评估

### 另类数据分类
| 类型 | 数据源 | 价值 | 时效性 |
|------|--------|------|--------|
| 新闻舆情 | 财经媒体/社交平台 | 情绪信号 | 分钟级 |
| 资金流向 | 交易所/券商 | 趋势信号 | 日级 |
| 行业报告 | 券商研究所 | 深度洞察 | 周/月级 |
| 政策公告 | 政府网站 | 事件驱动 | 即时 |
| 卫星数据 | 商业卫星公司 | 实物验证 | 周级 |

### 信息噪声过滤
```
信号 vs 噪声:

高信号: 
  - 央行货币政策调整
  - 上市公司正式公告
  - 行业监管政策变化
  - 大额资金持续流入/流出

高噪声:
  - 未经证实的传闻
  - 单日异常数据
  - 情绪化的社交媒体发言
  - 过度解读的"专家"观点

过滤方法:
  1. 多源交叉验证（至少2个独立源确认）
  2. 时间衰减加权（越新的权重越高，但不忽略历史）
  3. 来源可信度评级（官方>主流媒体>自媒体>传闻）
```
