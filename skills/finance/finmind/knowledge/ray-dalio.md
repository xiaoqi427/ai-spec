# 瑞·达利欧投资哲学 —《原则》精华

> **"痛苦 + 反思 = 进步"**

## 生活与工作原则（适用于投资）

### 极度透明（Radical Transparency）
- 对自己的分析逻辑保持100%透明
- 能清晰向他人解释每一步推理
- 如果不能解释清楚，说明你还没想清楚
- **FinMind应用**: 所有分析必须标注推理过程和数据来源

### 拥抱现实（Embrace Reality）
- 接受市场现实，不要对抗趋势
- 不要让愿望取代分析
- "真相——准确理解现实——是任何良好结果的根本依据"
- **FinMind应用**: 数据驱动决策，不做主观臆断

### 系统化决策（Systematize Decision Making）
- 把决策过程转化为可重复的系统
- 用原则而不是情绪来指导行动
- "如果你能把自己的原则写成算法，让计算机帮你决策，那就更好了"
- **FinMind应用**: 评分卡系统 + 检查清单 = 系统化决策

## 投资原则

### 全天候策略（All Weather）
```
核心理念: 不要试图预测经济环境，而是构建在任何环境下都能生存的组合

四种经济环境:
  1. 经济增长 > 预期 → 股票↑ 商品↑
  2. 经济增长 < 预期 → 债券↑ 黄金↑
  3. 通胀 > 预期 → 商品↑ 通胀保值债券↑
  4. 通胀 < 预期 → 股票↑ 债券↑

全天候配置参考:
  股票: 30%
  长期国债: 40%
  中期国债: 15%
  黄金: 7.5%
  大宗商品: 7.5%

核心: 风险平价（Risk Parity）— 让每种资产承担相同的风险
```

### 债务周期理论
```
短期债务周期（5-8年）:
  扩张 → 泡沫 → 紧缩 → 衰退 → 复苏 → 扩张

长期债务周期（50-75年）:
  大规模杠杆化 → 债务危机 → 去杠杆化
  
  去杠杆化的四种方式:
    1. 债务减免/违约
    2. 紧缩政策
    3. 财富转移（加税）
    4. 印钞/货币贬值
    
  美丽的去杠杆: 四种方式平衡使用
  丑陋的去杠杆: 过度依赖某一种方式
```

### 风险管理原则
1. **永远不要把所有鸡蛋放在一个篮子里** — 分散化
2. **为最坏的情况做准备** — 压力测试
3. **不要赌你承受不起的损失** — 仓位控制
4. **保持流动性** — 永远有现金储备
5. **对冲不确定性** — 不要裸奔

## 思维工具

### 五步流程
```
1. 设定目标 → "我要实现什么投资回报?"
2. 识别问题 → "什么阻碍我实现目标?"
3. 诊断问题 → "根本原因是什么?"
4. 设计方案 → "怎么解决?"
5. 执行方案 → "坚持执行，定期复盘"
```

### 可信度加权决策
- 不是所有意见都同等重要
- 在某个领域有良好记录的人的意见权重更高
- 多人独立分析，加权汇总
