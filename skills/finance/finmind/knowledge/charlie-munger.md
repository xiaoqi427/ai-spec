# 查理芒格思维模型

> **"如果我知道我会死在哪里，我就永远不去那个地方。"**

## 核心思维框架

### 多元思维模型（Mental Models）

芒格提倡从多个学科借鉴思维模型，避免"拿着锤子看什么都像钉子"。

#### 关键思维模型清单

| 编号 | 模型 | 学科 | 投资应用 |
|------|------|------|---------|
| M01 | **复利效应** | 数学 | 长期持有优质资产的回报远超想象 |
| M02 | **概率思维** | 数学 | 用期望值而非确定性来评估投资 |
| M03 | **会计学基础** | 会计 | 读懂财务报表，识别财务造假 |
| M04 | **微观经济学** | 经济 | 供需关系、竞争优势、规模效应 |
| M05 | **心理学偏差** | 心理 | 识别自己和市场的认知偏差 |
| M06 | **进化论** | 生物 | 企业适应环境的能力决定存亡 |
| M07 | **临界质量** | 物理 | 网络效应达到临界点后爆发增长 |
| M08 | **反馈循环** | 工程 | 正反馈(泡沫)与负反馈(均值回归) |

| M10 | **帕累托法则** | 统计 | 20%的持仓贡献80%的收益 |

### 能力圈（Circle of Competence）

```
核心原则:
  "知道自己知道什么，更重要的是知道自己不知道什么"
  
能力圈三层:
  内圈(深度理解): 可以重仓，有很高的确信度
  中圈(一般理解): 可以小仓位参与，持续学习
  外圈(不理解):   不碰，再便宜也不是你的机会

扩大能力圈的方法:
  1. 大量阅读（芒格每天阅读500页以上）
  2. 跨学科学习
  3. 从错误中学习（自己的和别人的）
  4. 与聪明人交流
```

### 反转思考（Inversion）

```
正向思维: "怎样才能投资成功?"
反转思维: "怎样才会投资失败? 然后避免它。"

投资中的反转思考:
  1. "什么会让这笔投资亏大钱?" → 列出至少3个风险场景
  2. "什么情况下这家公司会衰落?" → 护城河是否真的牢固
  3. "我可能遗漏了什么?" → 检查盲点
  4. "如果对手来做这个分析，他们会怎么反驳?" → 考虑反面证据
```

### 检查清单（Checklist）

```
芒格投资检查清单（简化版）:

□ 1. 我理解这家公司吗?（能力圈）
□ 2. 这家公司有持久的竞争优势吗?（护城河）
□ 3. 管理层是否诚实且有能力?（人的因素）
□ 4. 价格是否合理?（安全边际）
□ 5. 我考虑了反面论据吗?（反转思考）
□ 6. 我的判断是否受到了情绪影响?（心理偏差）
□ 7. 如果亏损50%，我能承受吗?（风险容忍度）
□ 8. 有没有更好的替代选择?（机会成本）
□ 9. 我的信息足够充分吗?（信息质量）
□ 10. 这笔投资符合我的长期策略吗?（一致性）
```

## 投资金句

- "投资很简单，但不容易。简单在于原则清晰，不容易在于知行合一"
- "大部分人不是被市场打败的，是被自己打败的"
- "集中投资几家真正理解的好公司，远好过分散投资一堆半懂不懂的公司"
- "耐心是投资者最重要的美德"
- "好的投资就是好的商业判断，加上自我约束"
- "如果一家公司不值得我持有很长时间，那它可能根本不值得我持有"
- "人类最大的错误不是在自己不知道的地方犯错，而是在自认为知道的地方犯错"
