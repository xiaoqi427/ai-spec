# 康波周期理论（Kondratieff Wave）

> **"人生发财靠康波。"** — 周金涛

## 理论概述

康波周期（康德拉季耶夫长波）是经济学中描述约50-60年为一个完整周期的长波理论。
每一轮长波由一项革命性技术创新驱动，经历从兴起到衰退的完整过程。

## 历史康波周期

| 波次 | 时间跨度 | 驱动技术 | 代表国家 | 核心资产 |
|------|---------|---------|---------|---------|
| 第1波 | 1780-1840 | 纺织/水力 | 英国 | 纺织业/运河 |
| 第2波 | 1840-1900 | 蒸汽/铁路 | 英国/美国 | 铁路/钢铁 |
| 第3波 | 1900-1950 | 电力/化工 | 美国/德国 | 电气/化工 |
| 第4波 | 1950-2000 | 汽车/计算机 | 美国/日本 | 汽车/半导体 |
| 第5波 | 2000-2050? | 信息技术/AI | 美国/中国 | 互联网/AI |

## 四阶段模型

```
一个完整康波周期:

繁荣期(Spring) → 衰退期(Summer) → 萧条期(Autumn) → 回升期(Winter)
   15-20年          约10年           约10年           10-15年

┌──────────┐   ┌──────────┐   ┌──────────┐   ┌──────────┐
│  繁荣期   │→ │  衰退期   │→ │  萧条期   │→ │  回升期   │
│          │   │          │   │          │   │          │
│ 技术爆发  │   │ 泡沫破裂  │   │ 产能出清  │   │ 新技术萌芽│
│ 投资热潮  │   │ 去杠杆化  │   │ 资产贬值  │   │ 需求恢复  │
│ 资产膨胀  │   │ 金融危机  │   │ 通缩压力  │   │ 周期重启  │
└──────────┘   └──────────┘   └──────────┘   └──────────┘
```

### 各阶段特征与资产配置

#### 繁荣期（Spring, ~15-20年）
- **经济特征**: GDP高增长、就业充分、科技创新加速
- **金融特征**: 信贷扩张、资产价格上涨、IPO热潮
- **最佳资产**: 股票(成长股) > 房地产 > 商品 > 债券
- **投资策略**: 积极做多，重仓权益资产

#### 衰退期（Summer, ~10年）
- **经济特征**: 增长放缓、通胀上升、产能过剩初现
- **金融特征**: 利率上升、泡沫膨胀、投机盛行
- **最佳资产**: 商品 > 房地产 > 现金 > 股票
- **投资策略**: 逐步防守，减少权益敞口，增加实物资产

#### 萧条期（Autumn, ~10年）
- **经济特征**: 经济衰退、失业上升、企业破产
- **金融特征**: 信贷收缩、资产暴跌、债务危机
- **最佳资产**: 现金 > 国债 > 黄金 > 股票
- **投资策略**: 现金为王，等待机会，抄底优质资产（末期）

#### 回升期（Winter, ~10-15年）
- **经济特征**: 触底复苏、新技术商业化、新产业崛起
- **金融特征**: 利率低位、信贷恢复、估值修复
- **最佳资产**: 股票(价值股) > 债券 > 房地产 > 商品
- **投资策略**: 从防守转向进攻，布局新技术/新产业

## 嵌套周期

康波并非孤立存在，内部嵌套着中短周期:

```
康波长波(50-60年)
  ├── 朱格拉周期(设备投资, 7-11年) × 5-6个
  │     ├── 基钦周期(库存, 3-5年) × 2-3个
  │     │     └── 季节性波动(1年) × 3-5个
  │     └── ...
  └── ...

判断当前位置:
  1. 先定位康波长波阶段（大方向）
  2. 再定位朱格拉中周期（中期趋势）
  3. 最后看基钦短周期（短期节奏）
  
三周期共振向上 → 最强做多信号
三周期共振向下 → 最强防守信号
```
