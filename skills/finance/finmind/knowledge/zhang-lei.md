# 张磊投资哲学 —《价值》核心方法论

> **"做时间的朋友。"**

## 核心投资理念

### 长期主义（Long-termism）
- "流水不争先，争的是滔滔不绝"
- "在正确的道路上，时间就是你的朋友"
- 不追求短期暴利，追求长期复合增长
- **FinMind应用**: 框架A的评估周期至少3年以上

### 研究驱动（Research-Driven）
- "研究创造价值"——深度研究是超额收益的来源
- 对行业和公司的理解深度决定投资回报
- "做别人不愿意做的功课，去别人不愿意去的地方"
- **FinMind应用**: 数据层的深度采集 + 研究层的多维分析

### 价值发现
- 寻找被市场低估的价值
- 不是价格便宜就有价值，而是"值得"的东西才有价值
- "投资最大的风险不是波动，而是永久性地损失本金"

## 投资方法论

### 三个维度看公司

| 维度 | 核心问题 | 评估要点 |
|------|---------|---------|
| **行业** | 赛道是否足够长? | 市场空间、增长趋势、竞争格局 |
| **公司** | 护城河是否足够深? | 商业模式、竞争优势、管理团队 |
| **管理层** | 人是否足够强? | 企业家精神、战略远见、执行力 |

### 动态护城河
```
传统护城河 vs 动态护城河:

传统: 一成不变的壁垒（品牌、牌照等）
动态: 持续进化的能力（创新、组织、文化）

张磊的动态护城河评估:
  1. 这家公司能不能持续创新? → 研发投入/新产品管线
  2. 组织能力是否跟得上增长? → 人才密度/管理效率
  3. 企业文化是否支持长期发展? → 价值观/使命感
  4. 是否有"疯狂创造价值"的企业家精神?
```

### 选人比选事更重要
- "投最好的企业家，让他们去选择做什么事"
- 优秀的管理层能把一般的行业做成好生意
- 平庸的管理层能把好行业做烂

### "长坡厚雪"模型
```
长坡: 行业空间够大，增长够持久（至少10年可见空间）
厚雪: 护城河够深，利润率够高（壁垒不易被攻破）

最佳组合: 长坡 + 厚雪 = 滚雪球效应
  示例: 消费品龙头（品牌+渠道+规模）
        平台型公司（网络效应+转换成本）
        
警惕: 长坡但薄雪 = 竞争激烈（如大多数制造业）
       厚雪但短坡 = 天花板明显（如某些细分领域龙头）
```

## 关于估值

- "好的投资不在于你买了什么，而在于你以什么价格买"
- "估值是一门艺术，不是精确的科学"
- "与其追求精确的错误，不如追求模糊的正确"
- 长期看，好公司的股价会反映其价值创造能力

## 关于风险

- "真正的风险不是波动，而是你不理解自己买的是什么"
- "最大的风险是你认为没有风险"
- "分散化可以降低非系统性风险，但无法消除系统性风险"
