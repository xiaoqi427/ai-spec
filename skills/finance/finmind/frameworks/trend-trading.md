# 框架B：趋势交易分析四象限（The Momentum Way）

> **"华尔街没有新鲜事，因为投机像群山一样古老。"** — 杰西·利弗莫尔
>
> 融合：《股票作手回忆录》利弗莫尔 + 康波周期 + 技术形态 + 游子/柚子实战体系

---

## 框架总览

```
Quadrant 1. 宏观周期定位 ──→ 顺周期/逆周期 判断
          │
Quadrant 2. 市场情绪测量 ──→ 恐贪指数/极端信号
          │
Quadrant 3. 技术形态研判 ──→ 趋势方向/关键位
          │
Quadrant 4. 交易计划制定 ──→ 入场/止损/止盈/仓位
```

**最终输出**: 趋势信号卡（买入/卖出/观望 + 入场位/止损位/目标位 + 仓位建议）

| 信号强度 | 条件 | 动作 |
|---------|------|------|
| 强买入 | 4象限全部看多 + 突破关键位 | 果断入场，按计划仓位 |
| 买入 | 3象限看多 + 1象限中性 | 分批入场，先建底仓 |
| 观望 | 信号矛盾或不明确 | 等待，不做模糊的交易 |
| 卖出 | 3象限看空 + 破止损位 | 利弗莫尔: "止损不犹豫" |
| 强卖出 | 4象限全部看空 | 清仓离场 |

## Quadrant 1: 宏观周期定位（Macro Cycle Positioning）

> **"人生发财靠康波。"** — 周金涛

### 1a. 康波长波周期（50-60年）

```
一个完整的康波周期:

繁荣期(约15-20年) → 衰退期(约10年) → 萧条期(约10年) → 回升期(约10-15年)
     ↑                                                        │
     └────────────────────────────────────────────────────────┘

历史康波周期:
  第1波(1780-1840): 纺织工业革命
  第2波(1840-1900): 蒸汽与铁路
  第3波(1900-1950): 电力与化工
  第4波(1950-2000): 汽车与计算机
  第5波(2000-2050?): 信息技术与AI

当前定位判断维度:
  - 技术革命所处阶段（AI是新周期的起点?）
  - 全球债务水平（高债务→萧条风险↑）
  - 人口结构趋势（老龄化→需求萎缩?）
  - 资源价格周期（大宗商品超级周期?）
```

### 1b. 朱格拉周期（设备投资，7-11年）

| 阶段 | 特征 | 资产表现 |
|------|------|---------|
| 扩张期 | 企业资本开支增加、产能扩张 | 股票↑ 商品↑ |
| 高峰期 | 产能过剩开始显现 | 股票见顶 商品↑ |
| 收缩期 | 投资减少、去产能 | 股票↓ 债券↑ |
| 低谷期 | 产能出清、触底反弹 | 股票底部 债券↑ |

### 1c. 基钦周期（库存，3-5年）

| 阶段 | 特征 | 操作含义 |
|------|------|---------|
| 主动补库存 | 需求↑ 库存↑ 价格↑ | 最佳做多阶段 |
| 被动补库存 | 需求↓ 库存↑ 价格→ | 信号开始转弱 |
| 主动去库存 | 需求↓ 库存↓ 价格↓ | 最佳防守阶段 |
| 被动去库存 | 需求↑ 库存↓ 价格→ | 底部信号，准备做多 |

### 1d. 宏观结论判断

```
三周期叠加判断:
  康波 + 朱格拉 + 基钦 = 综合周期定位

最强做多: 三周期同时处于上升阶段
最强防守: 三周期同时处于下降阶段
分歧期:   短周期上升 + 长周期下降 → 反弹而非反转，谨慎做多
```

### Quadrant 1 信号输出

| 信号 | 条件 |
|------|------|
| 🟢 顺周期做多 | 至少2个周期处于上升阶段 |
| 🟡 中性观望 | 周期信号矛盾 |
| 🔴 逆周期防守 | 至少2个周期处于下降阶段 |

## Quadrant 2: 市场情绪测量（Market Sentiment）

> **"市场由两种情绪驱动：恐惧和贪婪。"** — 利弗莫尔

### 2a. 恐贪指数

| 指标 | 极度恐惧(0-25) | 恐惧(25-45) | 中性(45-55) | 贪婪(55-75) | 极度贪婪(75-100) |
|------|----------------|-------------|-------------|-------------|-----------------|
| VIX | > 30 | 20-30 | 15-20 | 12-15 | < 12 |
| 涨跌家比 | < 0.3 | 0.3-0.5 | 0.5-0.7 | 0.7-0.9 | > 0.9 |
| 融资余额变化 | 大幅减少 | 减少 | 平稳 | 增加 | 大幅增加 |
| 新开户数 | 冰点 | 低迷 | 正常 | 活跃 | 疯狂 |

**反人性操作原则**（来自行为金融学）:
- 极度恐惧 = 别人恐惧时贪婪（候选买入区间）
- 极度贪婪 = 别人贪婪时恐惧（候选减仓区间）
- 但注意: "不要接下落的刀"——等趋势确认再行动

### 2b. 资金流向分析

| 指标 | 看多信号 | 看空信号 |
|------|---------|---------|
| 北向资金 | 连续3日净买入 | 连续3日净卖出 |
| 融资余额 | 稳步增加 | 急速下降 |
| ETF申赎 | 大额申购（机构入场） | 大额赎回 |
| 主力资金 | 大单净买入 | 大单净卖出 |

### 2c. 利弗莫尔"关键价位"情绪判断

> **"股票在关键价位的表现，比任何其他信息都重要。"** — 利弗莫尔

```
关键价位类型:
  - 历史前高（阻力位）
  - 历史前低（支撑位）
  - 整数关口（心理关口）
  - 缺口位置（跳空缺口）
  - 均线密集区

判断方法:
  在关键位放量突破 → 趋势加速信号（利弗莫尔最重要的买入信号）
  在关键位缩量受阻 → 趋势可能反转
  在关键位反复争夺 → 方向待定，等待突破
```

### 2d. 认知偏差扫描（引自 behavioral-finance.md）

在情绪分析阶段，必须扫描当前市场是否处于以下偏差陷阱:

| 偏差 | 市场表现 | 对策 |
|------|---------|------|
| **羊群效应** | 全民谈股票、出租车司机推荐股票 | 极度警惕，可能是顶部 |
| **锚定效应** | 紧盯着历史最高价/最低价 | 忽略锚点，看当下趋势 |
| **近因偏差** | 因为最近涨了就觉得会一直涨 | 拉长时间轴，看大周期 |
| **损失厌恶** | 亏损的仓位死扛不割 | 利弗莫尔: "止损是生存的代价" |

### Quadrant 2 信号输出

| 信号 | 条件 |
|------|------|
| 🟢 情绪偏暖利于做多 | 恐贪指数45-65 + 资金净流入 |
| 🟡 极端情绪需警惕 | 恐贪指数<25或>75 |
| 🔴 情绪极度亢奋 | 恐贪指数>80 + 散户疯狂入场 |

## Quadrant 3: 技术形态研判（Technical Analysis）

> **"不要和市场争论。"** — 利弗莫尔

### 3a. 趋势判断（最核心）

```
趋势判断三重验证:

1. 均线系统:
   MA5 > MA20 > MA60 > MA120 → 多头排列 → 上升趋势
   MA5 < MA20 < MA60 < MA120 → 空头排列 → 下降趋势
   均线交织 → 震荡/方向不明

2. MACD:
   DIF > DEA 且柱状线放大 → 上升动能增强
   DIF < DEA 且柱状线放大 → 下降动能增强
   金叉(DIF上穿DEA) → 买入信号
   死叉(DIF下穿DEA) → 卖出信号

3. 趋势线:
   价格沿上升趋势线运行 → 持有
   价格跌破上升趋势线 → 警惕
   价格突破下降趋势线 → 关注
```

### 3b. 经典形态识别

| 形态类型 | 形态名称 | 信号 | 可靠度 |
|---------|---------|------|--------|
| **反转** | 头肩顶 | 卖出 | ★★★★★ |
| **反转** | 头肩底 | 买入 | ★★★★★ |
| **反转** | 双顶(M顶) | 卖出 | ★★★★ |
| **反转** | 双底(W底) | 买入 | ★★★★ |
| **反转** | 圆弧底 | 买入 | ★★★★ |
| **持续** | 上升三角形 | 继续涨 | ★★★★ |
| **持续** | 下降三角形 | 继续跌 | ★★★★ |
| **持续** | 旗形整理 | 继续原趋势 | ★★★ |
| **持续** | 楔形收敛 | 即将选择方向 | ★★★ |

### 3c. 量价关系分析

| 量价组合 | 含义 | 操作含义 |
|---------|------|---------|
| **放量上涨** | 多方强势推动 | 趋势加速，持有/加仓 |
| **缩量上涨** | 上涨动力减弱 | 注意高位风险 |
| **放量下跌** | 空方强势打压 | 趋势恶化，减仓/止损 |
| **缩量下跌** | 卖压减弱 | 可能接近底部 |
| **放量突破关键位** | 利弗莫尔最重要信号 | 强买入/强卖出 |
| **缩量回踩支撑** | 正常回调，不改趋势 | 回调买入机会 |

### 3d. 关键位标注

```
必须标注的关键位:
  ├── 近期高点（阻力位）
  ├── 近期低点（支撑位）
  ├── MA20/MA60/MA120 均线位置
  ├── 前期成交密集区（筹码峰）
  ├── 跳空缺口
  └── 整数关口
```

### 3e. 游子/柚子实战体系

```
【选股法则】
  1. 强势行业: 选近期涨幅靠前的行业板块
  2. 龙头个股: 行业内涨幅最大、成交最活跃的个股
  3. 量价配合: 放量突破前高，回调不破关键支撑
  4. 题材催化: 有政策/事件催化（加分项）

【择时法则】
  买入时机:
    ├── A. 突破买入: 放量突破关键阻力位，当日或次日买入
    ├── B. 回踩买入: 突破后缩量回踩支撑位确认，在支撑位买入
    └── C. 首阴买入: 强势股首次收阴线，次日不破低则买入

  卖出时机:
    ├── A. 止损卖出: 跌破止损位，无条件执行
    ├── B. 止盈卖出: 达到目标位或出现放量滞涨
    └── C. 趋势卖出: 跌破上升趋势线或均线系统走坏

【仓位管理】
  ├── 试探仓: 20-30%（突破初期/不确定性较高）
  ├── 标准仓: 50%（趋势确认后加仓）
  ├── 重仓: 70-80%（强趋势+多信号共振）
  └── 清仓: 100%卖出（破止损/趋势反转）

  金字塔加仓法:
    第1次: 30% → 突破买入
    第2次: 20% → 回踩确认后加仓
    第3次: 10% → 再次突破新高后小额加仓
    ※ 越涨加仓越少，控制平均成本
```

### Quadrant 3 信号输出

| 信号 | 条件 |
|------|------|
| 🟢 技术面看多 | 多头排列 + 放量突破 + 无顶部形态 |
| 🟡 技术面中性 | 趋势不明或形态未完成 |
| 🔴 技术面看空 | 空头排列 + 放量下跌 + 顶部形态 |

## Quadrant 4: 交易计划制定（Trade Plan）

> **"在进入市场之前就确定你的退出策略。"** — 利弗莫尔

### 4a. 入场条件（必须同时满足）

```
标准入场条件:
  □ Quadrant 1 周期定位: 非逆周期 (🟢 或 🟡)
  □ Quadrant 2 市场情绪: 非极端亢奋 (非🔴)
  □ Quadrant 3 技术形态: 看多 (🟢)
  □ 满足游子/柚子选股三要素: 强势行业 + 龙头 + 量价配合

至少满足 3/4 项才能入场
全部满足 4/4 项 → 强买入信号
```

### 4b. 止损位设定

```
止损方法（三选一，取最适合的）:

方法1: 关键位止损
  止损位 = 最近关键支撑位 - 缓冲(1-2%)
  
方法2: ATR止损
  止损位 = 入场价 - 2×ATR(14日)
  
方法3: 百分比止损
  止损位 = 入场价 × (1 - 最大容忍亏损%)
  默认最大容忍: 5-8%

利弗莫尔原则:
  "止损是保险费。不买保险的人，终将付出更大的代价。"
  "永远不要移动止损位来扩大亏损。"
```

### 4c. 止盈策略

```
止盈方法（可组合使用）:

方法1: 目标位止盈
  目标位1 = 入场价 × (1 + 预期涨幅%)    → 减仓1/3
  目标位2 = 前期重要高点                   → 减仓1/3
  目标位3 = 延伸目标                      → 最后1/3

方法2: 移动止盈
  随着价格上涨，逐步上移止损位
  规则: 每上涨X%，止损上移至成本价/盈利位
  
方法3: 技术止盈
  出现以下信号时止盈:
  - 顶部反转形态（头肩顶/双顶）
  - 放量滞涨（量增价不增）
  - 均线系统死叉
```

### 4d. 仓位管理

```
单笔风险控制:
  单笔最大亏损 = 总资金 × 2%（铁律）
  
  仓位计算:
  仓位金额 = 单笔最大亏损金额 / (入场价 - 止损价) × 入场价
  
  示例:
    总资金: 100万
    入场价: 50元
    止损价: 47元 (跌6%)
    单笔最大亏损: 100万 × 2% = 2万
    最大买入: 2万 / (50-47) × 50 = 33.3万 → 33%仓位

组合仓位控制:
  - 单只个股最大仓位: 30%
  - 同行业最大仓位: 40%
  - 总持仓上限: 80%（保留20%现金）
  - 同时持有个股数: 3-5只
```

## 与框架A的关系

| 维度 | 框架A(价值投资) | 框架B(趋势交易) |
|------|----------------|----------------|
| 时间维度 | 3年以上 | 数周到数月 |
| 核心驱动 | 内在价值 | 市场动量 |
| 买入条件 | 好公司+好价格 | 好趋势+好位置 |
| 卖出条件 | 基本面恶化/严重高估 | 趋势反转/止损触发 |
| 持仓数量 | 集中(3-5只) | 灵活(1-8只) |
| 适合人格 | 耐心、独立思考 | 敏锐、纪律性强 |

**双框架交叉验证**（--framework both 模式）:
- 两套框架**同时看多** → 高度确信，可以重仓
- 价值看多 + 趋势看空 → 价值底仓，等趋势确认再加仓
- 价值看空 + 趋势看多 → 短线博弈，严格止损
- 两套框架**同时看空** → 坚决回避
