# 框架A：价值投资分析五步法（The Value Way）

> **"投资的本质是买入一家好公司的一部分。"** — 段永平
>
> 融合：段永平《大道》+ 查理芒格 + 张磊《价值》+ 瑞·达利欧《原则》+ 《财务报表分析与股票估值》

---

## 框架总览

```
Step 1. 能力圈判断 ────→ PASS/CONTINUE
          │
Step 2. 商业模式评估 ──→ 护城河评分(1-10)
          │
Step 3. 财务健康度检验 → 财务评分(1-10)
          │
Step 4. 估值与安全边际 → 估值评分(1-10)
          │
Step 5. 决策检查清单 ──→ 最终决策(买入/观望/PASS)
```

**最终输出**: 价值评分卡（5维度 × 10分 = 50分满分）

| 评分区间 | 结论 | 动作 |
|---------|------|------|
| 40-50 | 卓越 | 重仓候选，等待合理价格 |
| 35-39 | 优秀 | 可以买入，注意安全边际 |
| 25-34 | 一般 | 观察，不急于行动 |
| < 25 | 不合格 | PASS，不在能力圈内 |

---

## Step 1: 能力圈判断（Circle of Competence）

> **"知道自己不知道什么，比聪明更重要。"** — 查理芒格

### 核心问题（必须全部回答"是"才能继续）

| 编号 | 问题 | 知识源 |
|------|------|--------|
| 1.1 | 我能用一句话说清这家公司是做什么的吗？ | 段永平"看不懂的不做" |
| 1.2 | 我理解这个行业的竞争格局和商业逻辑吗？ | 芒格"能力圈" |
| 1.3 | 我知道这家公司的客户是谁、怎么赚钱吗？ | 张磊"商业洞察" |
| 1.4 | 如果不看股价，我愿意持有这家公司3年以上吗？ | 段永平"买股票=买公司" |
| 1.5 | 我能列出至少3个这家公司可能失败的原因吗？ | 芒格"反转思考" |

### 评分标准

| 分数 | 标准 |
|------|------|
| 9-10 | 完全理解，是自己深耕的行业 |
| 7-8 | 大部分理解，有明确的认知边界 |
| 5-6 | 一般理解，需要补充研究 |
| < 5 | 理解不足 → **建议PASS，不浪费时间在不懂的事情上** |

### 段永平金句提醒
- "不懂不做。不懂的意思是看不到10年以后。"
- "投资最重要的是不亏大钱，不是赚大钱。"
- "看不懂的东西，再便宜也不是你的机会。"

---

## Step 2: 商业模式评估（Business Model Assessment）

> **"长期看，股票回报率等于企业的ROE。"** — 查理芒格

### 2a. 护城河分析（Moat Analysis）

逐项评估，每项1-2分，满分10分：

| 护城河类型 | 评估要点 | 分值 | 知识源 |
|-----------|---------|------|--------|
| **品牌护城河** | 品牌溢价能力、消费者心智占有 | 0-2 | 段永平"消费者心中的差异化" |
| **网络效应** | 用户越多价值越大（平台型） | 0-2 | 张磊"动态护城河" |
| **转换成本** | 客户迁移成本高不高 | 0-2 | 芒格"客户锁定" |
| **规模优势** | 规模带来的成本优势 | 0-2 | 张磊"规模效应" |
| **特许经营/牌照** | 是否有政策壁垒 | 0-2 | 达利欧"监管环境" |

### 2b. 管理层评估

| 维度 | 评估要点 | 知识源 |
|------|---------|--------|
| **诚信度** | 历史承诺兑现率、信息披露质量 | 段永平"本分"：做对的事情 |
| **能力** | 过往业绩、战略执行力 | 张磊"选对人比选对事更重要" |
| **利益一致性** | 管理层持股、薪酬结构 | 芒格"激励机制决定行为" |
| **企业文化** | 创新vs官僚、长期vs短期 | 段永平"企业文化是最大的护城河" |

### 2c. 行业空间与竞争格局

| 维度 | 评估要点 | 知识源 |
|------|---------|--------|
| **赛道长度** | TAM(总可寻址市场)有多大 | 张磊"长坡厚雪" |
| **增长阶段** | 导入期/成长期/成熟期/衰退期 | 康波周期视角 |
| **竞争强度** | 波特五力分析 | 芒格"竞争优势" |
| **行业周期性** | 强周期/弱周期/消费 | 达利欧"周期认知" |

### Step 2 评分

| 分数 | 标准 |
|------|------|
| 9-10 | 超强护城河 + 优秀管理层 + 广阔赛道（如茅台级别） |
| 7-8 | 明确护城河 + 靠谱管理层 + 较好赛道 |
| 5-6 | 有一定壁垒但不够深，或管理层一般 |
| < 5 | 护城河浅、竞争激烈 → 不符合价值投资标准 |

---

## Step 3: 财务健康度检验（Financial Health Check）

> **"好公司的财务报表会'说话'，关键是你听得懂吗?"** — 《财务报表分析与股票估值》

### 3a. 盈利质量分析

| 指标 | 优秀 | 良好 | 警惕 | 来源 |
|------|------|------|------|------|
| **经营现金流/净利润** | > 100% | 80-100% | < 80% | 《财务报表分析》 |
| **净利率** | > 20% | 10-20% | < 10% | 《经营分析》 |
| **毛利率趋势** | 稳定或上升 | 小幅波动 | 持续下降 | 《经营分析》 |
| **应收账款周转天数** | 减少 | 稳定 | 增加 | 《财务报表分析》 |

**核心判断**: 赚的是"真金白银"还是"纸上富贵"?
- 经营现金流/净利润 < 80% → 盈利质量存疑，需要深挖原因

### 3b. 资产负债健康度

| 指标 | 优秀 | 良好 | 警惕 | 来源 |
|------|------|------|------|------|
| **资产负债率** | < 40% | 40-60% | > 60% | 《财务报表分析》 |
| **流动比率** | > 2.0 | 1.5-2.0 | < 1.5 | 《经营分析》 |
| **有息负债率** | < 20% | 20-40% | > 40% | 段永平"不借钱" |
| **商誉/总资产** | < 5% | 5-15% | > 15% | 《财务报表分析》 |

**段永平提醒**: "不借钱做生意的公司，在危机中往往能活下来。"

### 3c. 成长可持续性

| 指标 | 优秀 | 良好 | 警惕 | 来源 |
|------|------|------|------|------|
| **营收增速(3年CAGR)** | > 20% | 10-20% | < 10% | 张磊"成长性" |
| **ROE(连续3年)** | > 20% | 15-20% | < 15% | 芒格"长期ROE" |
| **ROIC vs WACC** | ROIC >> WACC | ROIC > WACC | ROIC ≈ WACC | 张磊"价值创造" |
| **自由现金流增速** | > 净利增速 | ≈ 净利增速 | < 净利增速 | 《财务报表分析》 |

### 3d. 杜邦分析拆解

```
ROE = 净利率 × 资产周转率 × 权益乘数

高ROE的三种模式:
├── 高净利率型（如茅台）: 好生意! 护城河带来的定价权
├── 高周转率型（如沃尔玛）: 效率驱动! 管理层执行力强
└── 高杠杆型（如银行）: 注意! 借钱赚钱有风险

最理想: 高净利率 + 合理周转 + 低杠杆
最危险: 低净利率 + 低周转 + 高杠杆 ← 段永平"远离这种公司"
```

### Step 3 评分

| 分数 | 标准 |
|------|------|
| 9-10 | 三项全优，现金流充沛，几乎无有息负债 |
| 7-8 | 主要指标优秀，个别指标良好 |
| 5-6 | 指标一般，有1-2个警惕信号 |
| < 5 | 多个警惕信号 → 财务风险较高 |

## Step 4: 估值与安全边际（Valuation & Margin of Safety）

> **"好公司 + 好价格 = 好投资。好公司但价格太贵 ≠ 好投资。"** — 段永平

### 4a. 绝对估值 — DCF 自由现金流折现

```
公司内在价值 = Σ (未来自由现金流 / (1+折现率)^n) + 终值

参数设定:
- 预测期: 10年
- 折现率(WACC): 通常8-12%
- 增长率假设: 基于Step3的成长性分析
- 终值增长率: 通常2-3%（不超过GDP增速）

输出: 每股内在价值估计区间 [保守值, 中性值, 乐观值]
```

### 4b. 相对估值 — 历史分位+同行比较

| 方法 | 操作 | 判断标准 |
|------|------|---------|
| **PE历史分位** | 与自身5年PE对比 | < 25%分位 = 低估, > 75%分位 = 高估 |
| **PB历史分位** | 与自身5年PB对比 | 同上 |
| **同行PE对比** | 与行业均值/龙头对比 | 低于同行 = 可能低估(或有原因) |
| **PEG** | PE/增长率 | < 1 = 合理, < 0.5 = 可能低估 |

### 4c. 安全边际计算

```
安全边际 = (内在价值 - 当前股价) / 内在价值 × 100%

段永平的出价法:
  "老巴说打四折买，我觉得至少要打七折以内才考虑"
  
安全边际要求:
  优质公司(高确定性): ≥ 20% 安全边际
  一般公司(中确定性): ≥ 30% 安全边际
  高不确定性公司: ≥ 40% 安全边际

段永平补充:
  "宁愿以合理价格买优秀公司，也不以便宜价格买平庸公司"
  → 对于真正的好公司，安全边际可以适度放宽
```

### Step 4 评分

| 分数 | 标准 |
|------|------|
| 9-10 | 显著低估（安全边际>30%），属于"恐惧时贪婪"的机会 |
| 7-8 | 合理偏低估（安全边际15-30%），可以开始建仓 |
| 5-6 | 合理估值（安全边际0-15%），持有可以，新买需谨慎 |
| 3-4 | 偏高估，等一等 |
| < 3 | 显著高估 → 段永平"再好的公司，价格太贵也不是好投资" |

## Step 5: 决策检查清单（Decision Checklist）

> **"使用检查清单可以避免许多愚蠢的错误。"** — 查理芒格

### 5a. 芒格检查清单（投资前必答）

| 编号 | 检查项 | 通过标准 |
|------|--------|---------|
| C01 | 我真的理解这家公司的商业模式吗？ | 能力圈内 |
| C02 | 这家公司10年后还会存在并且更强吗？ | 护城河可持续 |
| C03 | 管理层是否诚实、有能力、与股东利益一致？ | 管理层评估通过 |
| C04 | 当前价格是否提供了足够的安全边际？ | 安全边际≥20% |
| C05 | 我是否考虑了最坏的情况？ | 反转思考完成 |
| C06 | 如果这笔投资亏损50%，我能承受吗？ | 仓位合理 |
| C07 | 我的决策是否受到了情绪影响？ | 偏差检测通过 |
| C08 | 有没有我可能遗漏的重要信息？ | 信息充分 |

### 5b. 段永平三不原则

| 原则 | 检查 | 说明 |
|------|------|------|
| **不做空** | ✅/❌ | "做空是无限风险有限收益的事情" |
| **不借钱** | ✅/❌ | "不用margin，不借钱投资" |
| **不做不懂的** | ✅/❌ | "看不懂就不碰，机会永远都有" |

### 5c. 张磊"时间的朋友"检验

- "3年后这家公司会比现在更值钱吗？" → 是/否
- "这家公司是时间的朋友还是时间的敌人？" → 朋友/敌人
- "如果不能卖出，我愿意持有它吗？" → 愿意/不愿意

### 5d. 达利欧"极度透明"检验

- "我的分析逻辑能清晰地向别人解释吗？" → 是/否
- "如果我的分析被公开，我会觉得尴尬吗？" → 不会/会
- "我是否考虑了与我观点相反的证据？" → 是/否

### Step 5 评分

| 分数 | 标准 |
|------|------|
| 9-10 | 所有检查项通过，逻辑清晰，心理准备充分 |
| 7-8 | 绝大多数通过，个别需要注意 |
| 5-6 | 有2-3项未通过，需要重新审视 |
| < 5 | 多项未通过 → 不建议操作 |
