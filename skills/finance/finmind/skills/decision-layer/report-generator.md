# 研报生成器 (Report Generator)

> 决策层技能 — 将全链路分析整合为结构化可读报告

## 功能
将数据层、研究层、决策层的全部输出整合为结构化的研究报告，支持多种模板。

## 报告类型

| 类型 | 命令 | 篇幅 | 内容 |
|------|------|------|------|
| **深度研报** | `--type deep` | 3000-5000字 | 全链路完整分析 |
| **快速简报** | `--type brief` | 500-1000字 | 核心结论+关键数据 |
| **组合复盘** | `--type review` | 1000-2000字 | 持仓回顾+绩效分析 |
| **每日简报** | `--type daily` | 300-500字 | 市场概览+关注标的 |

## 深度研报结构

```
# {公司名}({代码}) 深度分析报告
日期: YYYY-MM-DD | 分析师: FinMind AI | 框架: {value/trend/both}

## 一、投资要点（Executive Summary）
  - 核心结论（一句话）
  - 操作建议 + 目标价
  - 关键风险

## 二、公司概况
  - 业务简介
  - 行业地位
  - 核心竞争力

## 三、财务分析
  - 盈利能力分析（含杜邦拆解）
  - 成长性分析
  - 资产负债健康度
  - 现金流质量

## 四、估值分析
  - 绝对估值（DCF）
  - 相对估值（PE/PB/PEG历史分位）
  - 安全边际计算

## 五、技术分析（如使用趋势框架）
  - 趋势判断
  - 形态识别
  - 关键位标注

## 六、风险提示
  - 芒格反转检验（3个风险场景）
  - 行业风险
  - 公司特有风险

## 七、投资评分卡
  （价值评分卡 和/或 趋势信号卡）

## 八、操作建议
  - 建议动作
  - 入场/止损/目标
  - 仓位建议

## 免责声明
  本报告仅供参考，不构成投资建议。
```

## 快速简报结构

```
# {公司名} 快速简报 | {日期}

💡 核心结论: {一句话}
🎯 建议: {买入/观望/卖出} | 仓位: {X%}
📊 评分: {X/50} | 置信度: {X%}

关键数据:
  价格: ¥{X} | PE: {X} | ROE: {X%} | 营收增速: {X%}

核心逻辑:
  1. {理由1}
  2. {理由2}
  3. {理由3}

主要风险:
  1. {风险1}
  2. {风险2}

⚠️ 仅供参考，不构成投资建议
```

## 报告质量要求

1. 所有数据必须标注来源和时效
2. 所有结论必须有推理过程支撑
3. 风险提示不能少于2条
4. 免责声明必须包含
5. 使用具体数字，避免模糊表述
