# 组合建议 (Portfolio Suggestion)

> 决策层技能 — 综合多维评分输出操作建议
> 知识融合: 段永平·集中持仓 + 芒格·检查清单 + 张磊·研究驱动

## 功能
综合研究层的评分、信号和风控检查结果，输出最终的操作建议（买入/卖出/持有/观望）。

## 决策矩阵

### 框架A（价值投资）决策规则
| 价值评分(50分制) | 偏差检测 | 决策 |
|-----------------|---------|------|
| ≥ 40 且 通过 | ✅ | 重仓候选，安全边际足够即买入 |
| 35-39 且 通过 | ✅ | 可以买入，注意安全边际 |
| 25-34 | 任意 | 观察，不急于行动 |
| < 25 | 任意 | PASS |

### 框架B（趋势交易）决策规则
| 看多象限数(4象限) | 偏差检测 | 决策 |
|------------------|---------|------|
| 4/4 且 通过 | ✅ | 强买入，按标准仓位入场 |
| 3/4 且 通过 | ✅ | 买入，先建底仓 |
| 2/4 | 任意 | 观望 |
| ≤ 1/4 | 任意 | 不操作/考虑卖出 |

### 双框架交叉验证（both模式）
| 框架A | 框架B | 综合决策 |
|-------|-------|---------|
| 买入 | 买入 | 高确信买入，仓位上限 |
| 买入 | 观望 | 价值底仓，等趋势确认再加 |
| 观望 | 买入 | 短线博弈，严格止损 |
| 观望/PASS | 观望/卖出 | 不操作 |

## 仓位计算
```
基础仓位 = 评分对应的标准仓位
  卓越(40+): 20-30%
  优秀(35-39): 10-20%
  一般(25-34): 0-5%（观察仓）

调整因子:
  × 市场环境系数（牛市1.2/震荡1.0/熊市0.7）
  × 确信度系数（高0.9-1.0/中0.7-0.8/低0.5-0.6）
  × 风控约束（不超过单股30%上限）

最终仓位 = min(基础仓位 × 调整因子, 风控上限)
```
