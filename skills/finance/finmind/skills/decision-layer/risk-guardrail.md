# 风控守门员 (Risk Guardrail)

> 决策层技能 — 实时监控风险指标，触发预警或否决
> 知识融合: 达利欧·全天候策略 + 利弗莫尔·止损纪律 + 行为偏差纠正

## 功能
作为决策层的"刹车"，对每一个操作建议进行风控检查，确保风险可控。

## 风控检查项

### 1. 仓位控制
| 规则 | 阈值 | 触发动作 |
|------|------|---------|
| 单只个股最大仓位 | ≤ 30% | 超标 → 否决/减仓 |
| 同行业最大仓位 | ≤ 40% | 超标 → 否决/减仓 |
| 总持仓上限 | ≤ 80% | 超标 → 不允许新增 |
| 最低现金保留 | ≥ 20% | 低于 → 不允许新增 |

### 2. 止损控制（利弗莫尔铁律）
| 规则 | 阈值 | 触发动作 |
|------|------|---------|
| 单笔最大亏损 | ≤ 总资金2% | 超标 → 必须缩小仓位 |
| 个股止损线 | ≤ -8% | 触发 → 无条件卖出 |
| 组合最大回撤 | ≤ -15% | 触发 → 全面减仓至50% |
| 连续止损次数 | ≤ 3次 | 触发 → 暂停交易，冷静复盘 |

### 3. 波动率监控
| 规则 | 阈值 | 触发动作 |
|------|------|---------|
| VIX > 30 | 市场恐慌 | 降低权益仓位至60% |
| 个股波动率 > 50% | 高风险标的 | 降低该股仓位上限至15% |

### 4. 认知偏差检查（反人性门禁）
- 是否存在"摊低成本"行为? → 否决加仓亏损仓位
- 是否连续3次交易同方向? → 提醒检查过度自信
- 是否在极端情绪下做决策? → 强制等待24小时

## 检查流程
```
输入: 操作建议(买入/卖出/加仓)
  ↓
仓位检查 → 通过/否决
  ↓
止损检查 → 通过/否决
  ↓
波动率检查 → 通过/调整仓位
  ↓
偏差检查 → 通过/警告
  ↓
输出: 通过✅ / 否决🚫 / 调整⚠️
```
