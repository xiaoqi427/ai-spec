# 康波周期分析 (Kondratieff Cycle Analyzer)

> 研究层技能 — 定位当前所处的康波周期阶段，给出资产配置建议
> 知识融合: 康波周期理论 + 周金涛预测框架

## 功能
基于宏观经济指标，判断当前处于康波周期的哪个阶段，并给出对应的资产配置建议。

## 分析维度

### 1. 三层周期定位
```
康波长波(50-60年) → 大方向
  ├── 判断指标: 技术革命阶段/全球债务水平/人口结构
  └── 当前输出: 繁荣/衰退/萧条/回升

朱格拉中周期(7-11年) → 中期趋势
  ├── 判断指标: 设备投资增速/产能利用率/企业资本开支
  └── 当前输出: 扩张/高峰/收缩/低谷

基钦短周期(3-5年) → 短期节奏
  ├── 判断指标: 库存水平/PPI/PMI
  └── 当前输出: 主动补库/被动补库/主动去库/被动去库
```

### 2. 三周期叠加判断
```
最强做多: 三周期同时上升 → 全力进攻
最强防守: 三周期同时下降 → 现金为王
分歧期:   短升长降 → 反弹而非反转，谨慎
```

### 3. 资产配置建议
基于周期阶段输出配置比例建议（参考达利欧全天候+周金涛框架）
