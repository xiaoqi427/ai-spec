# 事件冲击分析 (Event Impact Analyzer)

> 研究层技能 — 评估突发事件对持仓/标的的短期冲击
> 知识融合: 达利欧·债务周期与去杠杆化分析框架

## 功能
专门处理突发新闻（加息、地缘政治、政策变动等），量化评估特定事件对标的的短期冲击。

## 事件分类与冲击模型

| 事件类型 | 典型事件 | 冲击范围 | 持续时间 |
|---------|---------|---------|---------|
| **货币政策** | 加息/降息/降准 | 全市场 | 数周-数月 |
| **财政政策** | 减税/刺激计划/预算 | 相关行业 | 数月 |
| **监管政策** | 行业整顿/反垄断 | 特定行业 | 数月-数年 |
| **地缘政治** | 战争/制裁/贸易战 | 全市场+特定板块 | 不确定 |
| **公司事件** | 财报暴雷/管理层变动 | 个股 | 数天-数周 |
| **黑天鹅** | 疫情/自然灾害/金融危机 | 全市场 | 不确定 |

## 冲击评估框架（达利欧债务周期视角）

```
事件冲击评估五步法:

1. 事件定性: 利好/利空/中性?
2. 冲击量级: 轻微(1-3)/中等(4-6)/重大(7-10)?
3. 影响范围: 个股/行业/全市场?
4. 持续时间: 短期(<1周)/中期(1-3月)/长期(>3月)?
5. 二阶效应: 是否会引发连锁反应?
```
