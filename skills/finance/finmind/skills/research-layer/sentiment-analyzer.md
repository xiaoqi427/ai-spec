# 情绪分析 (Sentiment Analyzer)

> 研究层技能 — 利用NLP对新闻和社媒进行情绪评分，捕捉市场预期
> 知识融合: 行为金融学·认知偏差

## 功能
对另类数据采集器(alt-data-collector)输出的新闻/社媒数据进行情绪评分和市场温度测量。

## 分析维度

### 1. 单条情绪评分
对每条新闻/讨论进行 -1(极度负面) 到 +1(极度正面) 的评分。

### 2. 综合情绪指数
将多条评分加权汇总为综合情绪指数:
- 权重因子: 来源可信度 × 时效性 × 相关度
- 输出: -1.0 ~ +1.0 的综合得分

### 3. 恐贪指数
```
综合恐贪指数(0-100):
  0-25:  极度恐惧 → 可能是买入机会（逆向）
  25-45: 恐惧 → 关注底部信号
  45-55: 中性 → 正常市场
  55-75: 贪婪 → 注意风险
  75-100: 极度贪婪 → 可能是卖出信号（逆向）

构成指标:
  - VIX/波动率水平 (20%)
  - 涨跌家数比 (15%)
  - 融资余额变化 (15%)
  - 新闻情绪综合 (20%)
  - 社媒讨论热度 (15%)
  - 资金流向 (15%)
```

### 4. 认知偏差扫描
自动检测当前市场情绪中的偏差信号（引用 behavioral-finance.md）:
- 羊群效应: 全民讨论某只股票?
- 过度自信: 一致看多无人看空?
- 近因偏差: 仅因近期走势好就极度乐观?

```json
    "外资持续加仓"
  ],

}
```
