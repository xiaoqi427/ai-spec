# 技术形态识别 (Technical Pattern Recognizer)

> 研究层技能 — 自动识别图表中的技术形态和关键位
> 知识融合: 利弗莫尔·关键位理论 + 游子/柚子实战体系

## 功能
基于K线数据，自动识别技术形态、趋势方向、关键支撑阻力位和量价关系。

## 分析维度

### 1. 趋势判断
```
均线系统判断:
  MA5/MA20/MA60/MA120 排列关系
  
  多头排列: MA5>MA20>MA60>MA120 → 上升趋势 (🟢)
  空头排列: MA5<MA20<MA60<MA120 → 下降趋势 (🔴)
  交织: 均线纠缠 → 震荡 (🟡)

MACD辅助:
  DIF>DEA + 红柱放大 → 上升动能强
  DIF<DEA + 绿柱放大 → 下降动能强
  金叉 → 买入参考 / 死叉 → 卖出参考
```

### 2. 经典形态识别
| 形态 | 类型 | 信号 | 确认条件 |
|------|------|------|---------|
| 头肩顶 | 反转 | 卖出 | 跌破颈线+放量 |
| 头肩底 | 反转 | 买入 | 突破颈线+放量 |
| 双顶(M) | 反转 | 卖出 | 跌破颈线 |
| 双底(W) | 反转 | 买入 | 突破颈线 |
| 上升三角形 | 持续 | 看涨 | 突破上边+放量 |
| 下降三角形 | 持续 | 看跌 | 跌破下边 |
| 旗形 | 持续 | 原方向 | 突破旗面 |

### 3. 量价关系
| 组合 | 含义 | 信号 |
|------|------|------|
| 放量上涨 | 多方强势 | 🟢 持有/加仓 |
| 缩量上涨 | 动力减弱 | 🟡 注意风险 |
| 放量下跌 | 空方强势 | 🔴 减仓/止损 |
| 缩量下跌 | 卖压减弱 | 🟡 可能接近底部 |
| 放量突破关键位 | 利弗莫尔核心信号 | 🟢🟢 强信号 |

### 4. 关键位标注
自动标注: 近期高点/低点、均线位、缺口、整数关口、成交密集区

### 5. 游子/柚子体系应用
```
选股信号检查:
  □ 行业强度: 近5日行业涨幅排名前30%?
  □ 龙头地位: 行业内涨幅/成交额排名前3?
  □ 量价配合: 放量突破 + 回调缩量?
  □ 形态良好: 不存在顶部反转形态?
```
