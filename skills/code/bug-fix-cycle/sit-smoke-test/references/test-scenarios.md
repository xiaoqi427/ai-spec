# 常见报账单测试场景模板

## 场景1: 冒烟测试 (smoke)

**适用**: 所有报账单类型

**步骤**:
1. 打开报账单列表页
2. 检查页面加载正常（不白屏、不404、不跳登录页）
3. 检查查询条件区渲染
4. 检查数据表格渲染
5. 检查关键按钮可见（新建、查询、导出等）
6. 点击"查询"按钮，验证表格刷新

**预期结果**: 页面正常加载，所有关键元素可见

---

## 场景2: 新建-保存 (create-save)

**适用**: 所有报账单类型

**步骤**:
1. 打开报账单列表页
2. 点击"新建"按钮
3. 等待新建页面加载
4. 填写必填字段:
   - 报账单位: 选择一个公司
   - 报账人: 当前用户
   - 币种: CNY
   - 其他必填字段根据模板类型而定
5. 点击"保存"按钮
6. 验证保存成功提示

**预期结果**: 提示"保存成功"，页面显示已保存的报账单

---

## 场景3: 新建-保存-提交 (create-save-submit)

**适用**: 所有报账单类型

**步骤**: 在场景2基础上:
7. 点击"提交"按钮
8. 如有确认弹框，点击"确认"
9. 验证提交成功

**预期结果**: 提示"提交成功"，报账单状态变为"待审批"

---

## 场景4: 查看已有单据 (view-existing)

**适用**: 有数据的报账单类型

**步骤**:
1. 打开报账单列表页
2. 点击"查询"获取列表
3. 点击第一条数据
4. 验证详情页加载正常
5. 检查头信息展示
6. 检查行信息表格
7. 检查附件/影像区域

**预期结果**: 详情页所有区域正常展示

---

## 场景5: Bug 验证 (bug-verify)

**适用**: 修复 Bug 后的验证

**步骤**:
1. 根据 Bug 描述的复现步骤操作
2. 到达 Bug 触发点
3. 执行触发操作
4. 验证是否复现

**预期结果**: Bug 不再复现，功能正常

---

## 各模板特殊必填字段

### T001/T065 (差旅报账)
- 出差事由
- 出发地/目的地
- 出差日期

### T010 (采购报账)
- 供应商
- 合同号
- 发票号

### T044 (杂项采购)

- 费用类别
- 发票信息

### T047/T048/T049 (费用报账)
- 费用类别
- 费用说明

- 税种
- 税额

- 资产名称
- 资产金额

## 通用验证点

### 金额校验
- 行金额汇总 = 头金额
- 金额不能为负数
- 金额精度正确（小数位数）

### 状态校验
- 新建: 状态为"草稿"
- 保存后: 状态为"草稿"
- 提交后: 状态为"待审批"
- 审批通过: 状态为"已通过"

### 权限校验
- 只能查看自己创建的或分配的报账单
- 不同角色看到不同操作按钮
