## 风险应对策略说明

### 威胁（负面风险）应对策略

| 策略 | 说明 | 适用场景 |
|------|------|---------|
| **规避(Avoid)** | 消除威胁或保护项目免受影响 | 高概率高影响风险 |
| **转移(Transfer)** | 将影响转移给第三方 | 财务类风险（保险、外包） |
| **减轻(Mitigate)** | 降低概率和/或影响到可接受范围 | 中等风险 |
| **接受(Accept)** | 承认风险但不主动采取措施 | 低概率低影响风险 |

### 机会（正面风险）应对策略

| 策略 | 说明 |
|------|------|
| **开拓(Exploit)** | 确保机会一定实现 |
| **分享(Share)** | 与第三方共享机会 |
| **增强(Enhance)** | 提高机会发生的概率或影响 |
| **接受(Accept)** | 愿意在机会发生时利用它 |

## 使用说明

1. **识别阶段**: 通过头脑风暴、检查单、SWOT分析等方法识别风险，填入风险描述
2. **定性分析**: 评估每个风险的概率和影响，计算风险值，确定风险等级
3. **应对规划**: 为中高风险制定应对策略和具体措施，分配责任人
4. **监控更新**: 定期（建议每周/每两周）审查风险状态，更新风险登记册
